1. Overview & Responsibilities
HorizonPower handles all kinetic and electrical energy, industrial manufacturing, and vehicle transportation in the Horizon Corebreach modpack. It simulates the technological leap from hand-cranked mills to nuclear-powered metropolises.
Power Grids: Node-based energy distribution supporting kinetic (early game) and RF/Forge Energy (mid-to-late game).
Generators: 15 distinct power generation blocks, requiring specific fuels, environmental conditions, and maintenance.
Industry & Automation: 11 massive multi-block and single-block processing machines that smelt, refine, and assemble items automatically.
Vehicles: 20 distinct rideable entities spanning land, sea, and air. Vehicles possess realistic fuel consumption, acceleration curves, and inventory storage.
Infrastructure: 8 road and rail blocks that directly modify the friction and top speed coefficients of vehicles traveling over them.

package com.horizonpack.horizonpower.core;

import com.horizonpack.horizonpower.events.PowerEventHandlers;
import com.horizonpack.horizonpower.network.PowerPacketHandler;
import net.neoforged.bus.api.IEventBus;
import net.neoforged.fml.ModContainer;
import net.neoforged.fml.common.Mod;
import net.neoforged.fml.config.ModConfig;
import net.neoforged.fml.event.lifecycle.FMLCommonSetupEvent;
import net.neoforged.neoforge.common.NeoForge;

@Mod(HorizonPower.MODID)
public class HorizonPower {
    public static final String MODID = "horizonpower";
    public static final String VERSION = "1.0.0";

    public HorizonPower(IEventBus modEventBus, ModContainer modContainer) {
        modContainer.registerConfig(ModConfig.Type.COMMON, PowerConfig.COMMON_SPEC);
        modContainer.registerConfig(ModConfig.Type.CLIENT, PowerConfig.CLIENT_SPEC);

        PowerRegistries.BLOCKS.register(modEventBus);
        PowerRegistries.ITEMS.register(modEventBus);
        PowerRegistries.BLOCK_ENTITIES.register(modEventBus);
        PowerRegistries.ENTITY_TYPES.register(modEventBus);
        PowerRegistries.MENU_TYPES.register(modEventBus);
        PowerRegistries.CREATIVE_TABS.register(modEventBus);

        modEventBus.addListener(this::commonSetup);
        modEventBus.addListener(PowerRegistries::registerEntityAttributes);
        modEventBus.addListener(PowerPacketHandler::register);

        NeoForge.EVENT_BUS.register(PowerEventHandlers.class);
    }

    private void commonSetup(FMLCommonSetupEvent event) {
        event.enqueueWork(() -> {
            PowerRegistries.initCaches();
        });
    }
}

package com.horizonpack.horizonpower.blocks.cables;

import net.minecraft.core.BlockPos;
import net.minecraft.core.Direction;
import net.minecraft.world.item.context.BlockPlaceContext;
import net.minecraft.world.level.BlockGetter;
import net.minecraft.world.level.LevelAccessor;
import net.minecraft.world.level.block.Block;
import net.minecraft.world.level.block.EntityBlock;
import net.minecraft.world.level.block.entity.BlockEntity;
import net.minecraft.world.level.block.state.BlockState;
import net.minecraft.world.level.block.state.StateDefinition;
import net.minecraft.world.level.block.state.properties.BooleanProperty;
import net.minecraft.world.phys.shapes.CollisionContext;
import net.minecraft.world.phys.shapes.Shapes;
import net.minecraft.world.phys.shapes.VoxelShape;

public class BaseCableBlock extends Block implements EntityBlock {

    public static final BooleanProperty NORTH = BooleanProperty.create("north");
    public static final BooleanProperty EAST = BooleanProperty.create("east");
    public static final BooleanProperty SOUTH = BooleanProperty.create("south");
    public static final BooleanProperty WEST = BooleanProperty.create("west");
    public static final BooleanProperty UP = BooleanProperty.create("up");
    public static final BooleanProperty DOWN = BooleanProperty.create("down");

    private final int maxTransferRF;
    private final int maxTransferFluid;

    public BaseCableBlock(Properties properties, int maxTransferRF, int maxTransferFluid) {
        super(properties);
        this.maxTransferRF = maxTransferRF;
        this.maxTransferFluid = maxTransferFluid;
        this.registerDefaultState(this.stateDefinition.any()
            .setValue(NORTH, false).setValue(EAST, false).setValue(SOUTH, false)
            .setValue(WEST, false).setValue(UP, false).setValue(DOWN, false));
    }

    @Override
    protected void createBlockStateDefinition(StateDefinition.Builder<Block, BlockState> builder) {
        builder.add(NORTH, EAST, SOUTH, WEST, UP, DOWN);
    }

    @Override
    public BlockState getStateForPlacement(BlockPlaceContext context) {
        BlockGetter level = context.getLevel();
        BlockPos pos = context.getClickedPos();
        return this.defaultBlockState()
            .setValue(NORTH, canConnect(level, pos, Direction.NORTH))
            .setValue(EAST, canConnect(level, pos, Direction.EAST))
            .setValue(SOUTH, canConnect(level, pos, Direction.SOUTH))
            .setValue(WEST, canConnect(level, pos, Direction.WEST))
            .setValue(UP, canConnect(level, pos, Direction.UP))
            .setValue(DOWN, canConnect(level, pos, Direction.DOWN));
    }

    @Override
    public BlockState updateShape(BlockState state, Direction facing, BlockState facingState, LevelAccessor level, BlockPos currentPos, BlockPos facingPos) {
        BooleanProperty property = getPropertyForDirection(facing);
        return state.setValue(property, canConnect(level, currentPos, facing));
    }

    private boolean canConnect(BlockGetter level, BlockPos pos, Direction direction) {
        BlockState facingState = level.getBlockState(pos.relative(direction));
        return facingState.getBlock() instanceof BaseCableBlock; // Simplified. True implementation checks Capabilities.
    }

    private BooleanProperty getPropertyForDirection(Direction dir) {
        return switch (dir) {
            case NORTH -> NORTH; case EAST -> EAST; case SOUTH -> SOUTH;
            case WEST -> WEST; case UP -> UP; case DOWN -> DOWN;
        };
    }

    @Override
    public VoxelShape getShape(BlockState state, BlockGetter level, BlockPos pos, CollisionContext context) {
        VoxelShape shape = Block.box(6, 6, 6, 10, 10, 10);
        if (state.getValue(NORTH)) shape = Shapes.or(shape, Block.box(6, 6, 0, 10, 10, 6));
        if (state.getValue(SOUTH)) shape = Shapes.or(shape, Block.box(6, 6, 10, 10, 10, 16));
        if (state.getValue(EAST)) shape = Shapes.or(shape, Block.box(10, 6, 6, 16, 10, 10));
        if (state.getValue(WEST)) shape = Shapes.or(shape, Block.box(0, 6, 6, 6, 10, 10));
        if (state.getValue(UP)) shape = Shapes.or(shape, Block.box(6, 10, 6, 10, 16, 10));
        if (state.getValue(DOWN)) shape = Shapes.or(shape, Block.box(6, 0, 6, 10, 6, 10));
        return shape;
    }

    @Override
    public BlockEntity newBlockEntity(BlockPos pos, BlockState state) {
        return new com.horizonpack.horizonpower.blockentities.CableBlockEntity(pos, state, maxTransferRF);
    }
}

package com.horizonpack.horizonpower.blockentities;

import com.horizonpack.horizonpower.core.PowerRegistries;
import com.horizonpack.horizonpower.energy.HorizonEnergyStorage;
import net.minecraft.core.BlockPos;
import net.minecraft.nbt.CompoundTag;
import net.minecraft.world.level.Level;
import net.minecraft.world.level.block.entity.BlockEntity;
import net.minecraft.world.level.block.state.BlockState;

public class GeneratorBlockEntity extends BlockEntity {

    public final HorizonEnergyStorage energyStorage;
    private final int generationPerTick;
    private int fuelTicksRemaining = 0;

    public GeneratorBlockEntity(BlockPos pos, BlockState state, int generationPerTick, int capacity) {
        super(PowerRegistries.GENERATOR_BE.get(), pos, state);
        this.generationPerTick = generationPerTick;
        this.energyStorage = new HorizonEnergyStorage(capacity, 0, generationPerTick * 2);
    }

    public static void serverTick(Level level, BlockPos pos, BlockState state, GeneratorBlockEntity be) {
        if (be.fuelTicksRemaining > 0) {
            be.fuelTicksRemaining--;
            be.energyStorage.receiveEnergyInternal(be.generationPerTick, false);
            be.setChanged();
        }
        
        // Push energy to adjacent blocks
        if (be.energyStorage.getEnergyStored() > 0) {
            com.horizonpack.horizonpower.energy.PowerGridNode.distributeEnergy(level, pos, be.energyStorage);
        }
    }

    public void addFuel(int ticks) {
        this.fuelTicksRemaining += ticks;
        this.setChanged();
    }

    @Override
    protected void saveAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.saveAdditional(tag, registries);
        tag.putInt("FuelTicks", fuelTicksRemaining);
        tag.putInt("Energy", energyStorage.getEnergyStored());
    }

    @Override
    protected void loadAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.loadAdditional(tag, registries);
        this.fuelTicksRemaining = tag.getInt("FuelTicks");
        this.energyStorage.setEnergy(tag.getInt("Energy"));
    }
}

    private static final EntityDataAccessor<Float> FUEL_AMOUNT = SynchedEntityData.defineId(BaseVehicleEntity.class, EntityDataSerializers.FLOAT);
    
    protected float maxSpeed;
    protected float acceleration;
    protected float currentSpeed = 0f;
    protected float steeringAngle = 0f;
    protected float fuelConsumptionRate;
    protected String requiredAge;

    public BaseVehicleEntity(EntityType<?> type, Level level, float maxSpeed, float acceleration, float fuelConsumptionRate, String requiredAge) {
        super(type, level);
        this.maxSpeed = maxSpeed;
        this.acceleration = acceleration;
        this.fuelConsumptionRate = fuelConsumptionRate;
        this.requiredAge = requiredAge;
    }

    @Override
    protected void defineSynchedData(SynchedEntityData.Builder builder) {
        builder.define(FUEL_AMOUNT, 0f);
    }

    @Override
    public void tick() {
        super.tick();
        if (this.isVehicle() && this.getFuel() > 0) {
            handlePhysics();
            if (currentSpeed > 0.01f) {
                this.setFuel(this.getFuel() - fuelConsumptionRate);
            }
        } else {
            currentSpeed *= 0.9f; // Decelerate if out of fuel or empty
        }
        
        Vec3 forward = Vec3.directionFromRotation(0, this.getYRot());
        this.setDeltaMovement(forward.scale(currentSpeed));
        this.move(net.minecraft.world.entity.MoverType.SELF, this.getDeltaMovement());
    }

    protected void handlePhysics() {
        // Overridden by specific vehicle classes (Land, Sea, Air)
    }

    public float getFuel() { return this.entityData.get(FUEL_AMOUNT); }
    public void setFuel(float fuel) { this.entityData.set(FUEL_AMOUNT, fuel); }

    public void updateInputs(boolean forward, boolean backward, boolean left, boolean right) {
        if (forward && currentSpeed < maxSpeed) currentSpeed += acceleration;
        if (backward && currentSpeed > -maxSpeed / 2) currentSpeed -= acceleration;
        if (left) this.setYRot(this.getYRot() - 5f);
        if (right) this.setYRot(this.getYRot() + 5f);
    }

    @Override
    protected void readAdditionalSaveData(CompoundTag tag) {
        this.setFuel(tag.getFloat("Fuel"));
    }

    @Override
    protected void addAdditionalSaveData(CompoundTag tag) {
        tag.putFloat("Fuel", this.getFuel());
    }

package com.horizonpack.horizonpower.energy;

import net.minecraft.nbt.CompoundTag;
import net.neoforged.neoforge.energy.IEnergyStorage;

public class HorizonEnergyStorage implements IEnergyStorage {

    private int energy;
    private final int capacity;
    private final int maxReceive;
    private final int maxExtract;

    public HorizonEnergyStorage(int capacity, int maxReceive, int maxExtract) {
        this.capacity = capacity;
        this.maxReceive = maxReceive;
        this.maxExtract = maxExtract;
    }

    @Override
    public int receiveEnergy(int maxReceive, boolean simulate) {
        if (!canReceive()) return 0;
        int energyReceived = Math.min(capacity - energy, Math.min(this.maxReceive, maxReceive));
        if (!simulate) {
            energy += energyReceived;
        }
        return energyReceived;
    }

    public int receiveEnergyInternal(int maxReceive, boolean simulate) {
        int energyReceived = Math.min(capacity - energy, maxReceive);
        if (!simulate) {
            energy += energyReceived;
        }
        return energyReceived;
    }

    @Override
    public int extractEnergy(int maxExtract, boolean simulate) {
        if (!canExtract()) return 0;
        int energyExtracted = Math.min(energy, Math.min(this.maxExtract, maxExtract));
        if (!simulate) {
            energy -= energyExtracted;
        }
        return energyExtracted;
    }

    public int extractEnergyInternal(int maxExtract, boolean simulate) {
        int energyExtracted = Math.min(energy, maxExtract);
        if (!simulate) {
            energy -= energyExtracted;
        }
        return energyExtracted;
    }

    @Override
    public int getEnergyStored() {
        return energy;
    }

    public void setEnergy(int energy) {
        this.energy = Math.max(0, Math.min(capacity, energy));
    }

    @Override
    public int getMaxEnergyStored() {
        return capacity;
    }

    @Override
    public boolean canExtract() {
        return this.maxExtract > 0;
    }

    @Override
    public boolean canReceive() {
        return this.maxReceive > 0;
    }

    public CompoundTag serializeNBT() {
        CompoundTag tag = new CompoundTag();
        tag.putInt("Energy", energy);
        return tag;
    }

    public void deserializeNBT(CompoundTag nbt) {
        energy = nbt.getInt("Energy");
    }
}
9.2 PowerGridNode
A simplified DFS distributor that pushes energy from generators through valid cable blocks into receiving machines.
File: energy/PowerGridNode.java
code

package com.horizonpack.horizonpower.energy;

import net.minecraft.core.BlockPos;
import net.minecraft.core.Direction;
import net.minecraft.world.level.Level;
import net.minecraft.world.level.block.entity.BlockEntity;
import net.neoforged.neoforge.capabilities.Capabilities;
import net.neoforged.neoforge.energy.IEnergyStorage;

import java.util.ArrayDeque;
import java.util.HashSet;
import java.util.Queue;
import java.util.Set;

public class PowerGridNode {

    public static void distributeEnergy(Level level, BlockPos sourcePos, HorizonEnergyStorage sourceStorage) {
        if (sourceStorage.getEnergyStored() <= 0) return;

        Set<BlockPos> visited = new HashSet<>();
        Queue<BlockPos> queue = new ArrayDeque<>();
        
        visited.add(sourcePos);
        for (Direction dir : Direction.values()) {
            queue.add(sourcePos.relative(dir));
        }

        int energyToDistribute = sourceStorage.extractEnergyInternal(sourceStorage.getEnergyStored(), true);
        if (energyToDistribute == 0) return;

        while (!queue.isEmpty() && energyToDistribute > 0) {
            BlockPos current = queue.poll();
            if (visited.contains(current)) continue;
            visited.add(current);

            IEnergyStorage handler = level.getCapability(Capabilities.EnergyStorage.BLOCK, current, null);
            if (handler != null && handler.canReceive()) {
                int accepted = handler.receiveEnergy(energyToDistribute, false);
                if (accepted > 0) {
                    energyToDistribute -= accepted;
                    sourceStorage.extractEnergyInternal(accepted, false);
                    if (level.getBlockEntity(current) != null) {
                        level.getBlockEntity(current).setChanged();
                    }
                }
            }

            // If it's a cable, keep traversing
            if (level.getBlockState(current).getBlock() instanceof com.horizonpack.horizonpower.blocks.cables.BaseCableBlock) {
                for (Direction dir : Direction.values()) {
                    queue.add(current.relative(dir));
                }
            }
        }
    }
}

    public static void serverTick(Level level, BlockPos pos, BlockState state, MachineBlockEntity be) {
        boolean isWorking = false;

        if (be.hasValidRecipe() && be.energyStorage.getEnergyStored() >= be.energyPerTick) {
            be.energyStorage.extractEnergyInternal(be.energyPerTick, false);
            be.progress++;
            isWorking = true;

            if (be.progress >= be.maxProgress) {
                be.processRecipe();
                be.progress = 0;
            }
        } else {
            if (be.progress > 0) be.progress--;
        }

        // Update block state lit/active boolean if needed
        boolean wasWorking = state.getValue(com.horizonpack.horizonpower.blocks.machines.BaseMachineBlock.ACTIVE);
        if (isWorking != wasWorking) {
            level.setBlock(pos, state.setValue(com.horizonpack.horizonpower.blocks.machines.BaseMachineBlock.ACTIVE, isWorking), 3);
        }
        
        if (isWorking) be.setChanged();
    }

23. Critical Implementation Notes
Capability Invalidation Caching: Because energy grids scan hundreds of blocks rapidly, querying level.getCapability(Capabilities.EnergyStorage.BLOCK, pos) every tick is extremely expensive. CableBlockEntity MUST implement BlockCapabilityCache pointing to its neighbors and update that cache dynamically when blocks update.
Fluid vs Energy Constraints: The BaseCableBlock determines functionality based on its defined transfer caps during registration. If maxTransferRF is 0, the block does not provide IEnergyStorage. If maxTransferFluid is 0, it does not provide IFluidHandler. Do not register capabilities for components a pipe shouldn't have.
Vehicle Entity Desync: When the player sends VehicleInputPacket, the server modifies the vehicle's currentSpeed and steeringAngle. To prevent violent client-side rubberbanding, the Client vehicle entity must predictively update its own vectors while waiting for the server's sync tick to correct drift.
Road Block Friction Application: In vanilla, Block#getFriction() is meant for slipping on ice. BaseRoadBlock does not override this. Instead, BaseVehicleEntity reads the BaseRoadBlock instance beneath it and modifies its internal maxSpeed variable explicitly, allowing cars to drive fast without sliding uncontrollably like they are on blue ice.
Energy Node Loop Prevention: PowerGridNode.distributeEnergy() uses a HashSet<BlockPos> visited queue. This guarantees that infinite loops do not happen in circular cable networks. However, large loops can lag the server. Limit the queue depth to 256 cable blocks. If power hasn't found a destination by then, it dissipates as heat loss.
Generator Fuel Gating via Age: GeneratorBlockEntity checks the HorizonAge stored in HorizonCore via the RequiredAge string. If a player in the Stone Age attempts to use a Coal Generator, it simply refuses to process fuel and throws a translated chat error.
Machine Automation Pipeline: MachineBlockEntity automatically pulls from the inventory of the block entity above it and pushes to the block entity below it, provided they are instances of IItemHandler. This removes the necessity of complex filtering pipes early-game, letting gravity do the work.
VoxelShape Connectivity: BaseCableBlock constructs a dynamic VoxelShape based on 6 boolean BlockState properties (NORTH, EAST, etc.). These properties update via updateShape when neighboring blocks change. The visual block model must utilize multipart JSONs (as seen in §16) to correctly render the connections, otherwise the cables will look like solid invisible cubes with floating textures.
