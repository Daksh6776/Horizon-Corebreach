Critical Implementation Notes
1. Overview & Responsibilities
HorizonSurvival transforms the player's biological existence into a deeply simulated mechanic. It removes the vanilla hunger bar, replacing it with a macro and micro-nutrient metabolic system.
Nutrition: Tracks 22 distinct nutrients. Imbalances lead to 18 specific deficiency diseases (interfacing with HorizonBiome's disease engine).
Food Spoilage & Preservation: Food dynamically rots over time based on ambient temperature and preservation methods (10 methods, from drying to irradiation).
Cooking & Culinary: Cooking alters nutrient bioavailability. 10 specific cooking techniques unlocking through the ages.
Medicine & Healthcare: Injuries and diseases require specific treatments (11 procedures) administered at specialized medical buildings (8 tiers).
Metabolism: Digestion operates over time. Calories provide energy for actions, proteins heal wounds, vitamins prevent long-term degradation.

@Mod(HorizonSurvival.MODID)
public class HorizonSurvival {
    public static final String MODID = "horizonsurvival";
    public static final String VERSION = "1.0.0";

    public HorizonSurvival(IEventBus modEventBus, ModContainer modContainer) {
        modContainer.registerConfig(ModConfig.Type.COMMON, SurvivalConfig.COMMON_SPEC);
        modContainer.registerConfig(ModConfig.Type.CLIENT, SurvivalConfig.CLIENT_SPEC);

        SurvivalRegistries.BLOCKS.register(modEventBus);
        SurvivalRegistries.ITEMS.register(modEventBus);
        SurvivalRegistries.BLOCK_ENTITIES.register(modEventBus);
        SurvivalRegistries.MENU_TYPES.register(modEventBus);

        SurvivalRegistries.NUTRIENT_DEFINITIONS.register(modEventBus);
        SurvivalRegistries.PRESERVATION_METHODS.register(modEventBus);
        SurvivalRegistries.COOKING_METHODS.register(modEventBus);
        SurvivalRegistries.MEDICAL_TREATMENTS.register(modEventBus);
        SurvivalDataComponents.DATA_COMPONENT_TYPES.register(modEventBus);

        modEventBus.addListener(this::commonSetup);
        modEventBus.addListener(SurvivalRegistries::registerNewRegistries);
        modEventBus.addListener(SurvivalPacketHandler::register);

        NeoForge.EVENT_BUS.register(SurvivalEventHandlers.class);
    }

    private void commonSetup(FMLCommonSetupEvent event) {
        event.enqueueWork(() -> {
            DigestionEngine.init();
            SpoilageEngine.init();
            HealthcareEngine.init();
        });
    }
}

package com.horizonpack.horizonsurvival.systems;

import com.horizonpack.horizonsurvival.data.components.FoodPropertiesComponent;
import com.horizonpack.horizonsurvival.data.components.SurvivalDataComponents;
import com.horizonpack.horizonsurvival.data.registry.PreservationMethod;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.world.item.ItemStack;
import net.minecraft.world.item.Items;

public class SpoilageEngine {

    public static final long BASE_SPOIL_TICKS = 144000; // 6 in-game days

    public static void init() {}

    /**
     * Evaluates an item stack. If it has a FoodPropertiesComponent and has passed its expiration tick, it turns to rot.
     * Called when item is moved in inventory or interacts with the world.
     */
    public static ItemStack evaluateSpoilage(ServerLevel level, ItemStack stack) {
        if (!stack.has(SurvivalDataComponents.FOOD_PROPERTIES.get())) return stack;

        FoodPropertiesComponent data = stack.get(SurvivalDataComponents.FOOD_PROPERTIES.get());
        if (data == null) return stack;

        long currentTick = level.getGameTime();
        long age = currentTick - data.creationTick();

        PreservationMethod method = getPreservationMethod(level, data.preservationMethodId());
        float multiplier = method != null ? method.spoilTimeMultiplier() : 1.0f;

        // Climate modifies spoilage (heat = faster, freezing = slower)
        float climateModifier = 1.0f; // Integrates with HorizonWorld.ClimateEngine

        if (age > (BASE_SPOIL_TICKS * multiplier * climateModifier)) {
            ItemStack rot = new ItemStack(Items.ROTTEN_FLESH, stack.getCount());
            // Fire FoodSpoiledEvent
            net.neoforged.neoforge.common.NeoForge.EVENT_BUS.post(new com.horizonpack.horizonsurvival.events.custom.FoodSpoiledEvent(stack, rot));
            return rot;
        }

        return stack;
    }

    private static PreservationMethod getPreservationMethod(ServerLevel level, ResourceLocation id) {
        return level.registryAccess().registryOrThrow(com.horizonpack.horizonsurvival.core.SurvivalRegistries.PRESERVATION_KEY).get(id);
    }
}

     */
    public static void tickMetabolism(ServerPlayer player) {
        IHorizonPlayerData data = player.getCapability(HorizonCapabilities.PLAYER_DATA).orElse(null);
        if (data == null) return;

        float bmr = 0.05f; // Base calories per tick
        if (player.isSprinting()) bmr *= 3.0f;
        if (player.isSwimming()) bmr *= 2.5f;

        // Weather integration: shivering burns calories
        if (data.getBodyTemperature() < 36.0f) bmr *= 1.5f;

        data.setCalories(data.getCalories() - bmr);
        data.consumeNutrients(0.005f); // Micro-nutrient decay

        // Check for deficiencies
        checkDeficiencies(player, data);
    }

    private static void checkDeficiencies(ServerPlayer player, IHorizonPlayerData data) {
        // Interacts directly with HorizonBiome DiseaseEngine
        if (data.getVitamin(com.horizonpack.horizoncore.data.VitaminType.C) < 5.0f) {
            // Apply Scurvy
        }
        if (data.getProtein() < 10.0f) {
            player.addEffect(new net.minecraft.world.effect.MobEffectInstance(net.minecraft.world.effect.MobEffects.WEAKNESS, 200, 1));
        }
        if (data.getHydration() < 10.0f) {
            player.hurt(player.damageSources().dryOut(), 1.0f);
        }
    }
}

    }

import com.horizonpack.horizonsurvival.core.SurvivalRegistries;

import net.minecraft.core.BlockPos;
import net.minecraft.nbt.CompoundTag;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.world.MenuProvider;
import net.minecraft.world.entity.player.Inventory;
import net.minecraft.world.entity.player.Player;
import net.minecraft.world.inventory.AbstractContainerMenu;
import net.minecraft.world.item.ItemStack;
import net.minecraft.world.level.Level;
import net.minecraft.world.level.block.entity.BlockEntity;
import net.minecraft.world.level.block.state.BlockState;
import net.neoforged.neoforge.items.ItemStackHandler;
import org.jetbrains.annotations.Nullable;

public class PreservationStationBlockEntity extends BlockEntity implements MenuProvider {

    public final ItemStackHandler inventory = new ItemStackHandler(2) {
        @Override
        protected void onContentsChanged(int slot) {
            setChanged();
        }
    };

    private int processTick = 0;
    private final int MAX_PROCESS_TICKS = 400;
    private final String targetMethod;

    public PreservationStationBlockEntity(BlockPos pos, BlockState state, String targetMethod) {
        super(SurvivalRegistries.BLOCK_ENTITIES.getEntries().stream()
                .filter(r -> r.getId().getPath().equals("preservation_station")).findFirst().get().get(), pos, state);
        this.targetMethod = targetMethod;
    }

    public static void serverTick(Level level, BlockPos pos, BlockState state, PreservationStationBlockEntity be) {
        ItemStack input = be.inventory.getStackInSlot(0);
        if (!input.isEmpty() && input.has(SurvivalDataComponents.FOOD_PROPERTIES.get())) {
            be.processTick++;
            if (be.processTick >= be.MAX_PROCESS_TICKS) {
                // Apply preservation
                ItemStack output = input.copy();
                output.setCount(1);
                
                FoodPropertiesComponent oldComp = output.get(SurvivalDataComponents.FOOD_PROPERTIES.get());
                FoodPropertiesComponent newComp = new FoodPropertiesComponent(
                        level.getGameTime(), // Reset expiration on preservation
                        ResourceLocation.fromNamespaceAndPath("horizonsurvival", be.targetMethod),
                        oldComp.cookingMethodId(),
                        oldComp.nutrientMap()
                );
                output.set(SurvivalDataComponents.FOOD_PROPERTIES.get(), newComp);

                if (be.inventory.insertItem(1, output, false).isEmpty()) {
                    be.inventory.extractItem(0, 1, false);
                }
                be.processTick = 0;
            }
            be.setChanged();
        } else {
            be.processTick = 0;
        }
    }

    @Override
    protected void saveAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.saveAdditional(tag, registries);
        tag.put("Inventory", inventory.serializeNBT(registries));
        tag.putInt("ProcessTick", processTick);
    }

    @Override
    protected void loadAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.loadAdditional(tag, registries);
        if (tag.contains("Inventory")) inventory.deserializeNBT(registries, tag.getCompound("Inventory"));
        processTick = tag.getInt("ProcessTick");
    }

    @Override
    public net.minecraft.network.chat.Component getDisplayName() {
        return net.minecraft.network.chat.Component.translatable("block.horizonsurvival.preservation_station");
    }

    @Nullable
    @Override
    public AbstractContainerMenu createMenu(int containerId, Inventory inventory, Player player) {
        return null; // Gui implementation
    }
}

import net.minecraft.resources.ResourceLocation;

@Mod.EventBusSubscriber(modid = HorizonSurvival.MODID)
public class SurvivalEventHandlers {

    @SubscribeEvent
    public static void onPlayerTick(PlayerTickEvent.Post event) {
        if (!event.getEntity().level().isClientSide && event.getEntity() instanceof ServerPlayer player) {
            if (player.tickCount % 20 == 0) { // Tick metabolism every 1 second
                DigestionEngine.tickMetabolism(player);
            }
        }
    }

    @SubscribeEvent
    public static void onItemPickup(EntityItemPickupEvent event) {
        if (!event.getEntity().level().isClientSide) {
            ItemStack stack = event.getItem().getItem();
            // Instantly check and swap for rotten flesh if spoiled
            ItemStack checked = SpoilageEngine.evaluateSpoilage((net.minecraft.server.level.ServerLevel)event.getEntity().level(), stack);
            if (checked.getItem() != stack.getItem()) {
                event.getItem().setItem(checked);
            }
        }
    }

    @SubscribeEvent
    public static void onFoodEaten(LivingEntityUseItemEvent.Finish event) {
        if (event.getEntity() instanceof ServerPlayer player && !player.level().isClientSide) {
            ItemStack stack = event.getItem();
            if (stack.has(SurvivalDataComponents.FOOD_PROPERTIES.get())) {
                FoodPropertiesComponent foodData = stack.get(SurvivalDataComponents.FOOD_PROPERTIES.get());
                IHorizonPlayerData playerData = player.getCapability(HorizonCapabilities.PLAYER_DATA).orElse(null);
                
                if (playerData != null && foodData != null) {
                    // Extract custom nutrients
                    for (Map.Entry<ResourceLocation, Float> entry : foodData.nutrientMap().entrySet()) {
                        // Resolve NutrientType from string ID (simplified)
                        // HorizonCore.addNutrient(playerData, entry.getKey(), entry.getValue());
                    }
                    playerData.setCalories(playerData.getCalories() + 200.0f); // Base calorie addition
                }
            }
        }
    }

package com.horizonpack.horizonsurvival.network.packets;

import com.horizonpack.horizonsurvival.core.HorizonSurvival;
import com.horizonpack.horizonsurvival.systems.HealthcareEngine;
import io.netty.buffer.ByteBuf;
import net.minecraft.network.codec.ByteBufCodecs;
import net.minecraft.network.codec.StreamCodec;
import net.minecraft.network.protocol.common.custom.CustomPacketPayload;
import net.minecraft.resources.ResourceLocation;
import net.neoforged.neoforge.network.handling.IPayloadContext;

import java.util.UUID;

public record ApplyTreatmentPacket(UUID patientId, ResourceLocation treatmentId) implements CustomPacketPayload {

    public static final Type<ApplyTreatmentPacket> TYPE = new Type<>(ResourceLocation.fromNamespaceAndPath(HorizonSurvival.MODID, "apply_treatment"));

    public static final StreamCodec<ByteBuf, ApplyTreatmentPacket> STREAM_CODEC = StreamCodec.composite(
        ByteBufCodecs.UUID, ApplyTreatmentPacket::patientId,
        ResourceLocation.STREAM_CODEC, ApplyTreatmentPacket::treatmentId,
        ApplyTreatmentPacket::new
    );

    @Override
    public Type<ApplyTreatmentPacket> type() { return TYPE; }

    public static void handleServer(ApplyTreatmentPacket packet, IPayloadContext context) {
        context.enqueueWork(() -> {
            net.minecraft.server.level.ServerPlayer doctor = (net.minecraft.server.level.ServerPlayer) context.player();
            net.minecraft.world.entity.player.Player patient = doctor.level().getPlayerByUUID(packet.patientId());
            
            if (patient != null) {
                // Facility bonus assumes patient is within range of the BlockEntity
                HealthcareEngine.applyTreatment(doctor.serverLevel(), patient, doctor, packet.treatmentId(), 1.0f);
            }
        });
    }
}

26. Critical Implementation Notes
Vanilla Hunger Bar Override:
Vanilla FoodData is deeply hardcoded into Player. To override it, SurvivalEventHandlers must cancel FoodLevelChangeEvent entirely and suppress vanilla starvation damage in LivingHurtEvent. The client-side HUD overlay representing the hunger bar is disabled via RenderGuiLayerEvent.Pre in the client package, replacing it with the HorizonCore nutrition HUD.
Real-time Spoilage vs TPS Drop:
Checking the FoodPropertiesComponent of every item stack in the world and in every inventory every tick will crash a server. SpoilageEngine.evaluateSpoilage is only called dynamically upon interaction: moving the item, picking it up, or preparing to eat it. Background rotting in chests is checked sparsely by overriding block entity ticks or checking timestamp deltas when the inventory is opened.
Data Component Safety in Crafting:
When cooking raw meat (e.g., in a CookingStationBlockEntity), the FoodPropertiesComponent from the input must be mathematically transformed, not destroyed. The output stack inherits the creationTick, but the cookingMethodId is overwritten, and the nutrientMap values are multiplied by the CookingMethod's bioavailability index.
Surgical Failure Math:
HealthcareEngine.applyTreatment relies on the skill value queried from IHorizonPlayerData. If the administering player's Medical skill is below the threshold for surgery_advanced (75), the effectiveHeal is inverted using a logarithmic scale, causing massive immediate damage (representing a surgical botch/hemorrhage) instead of healing.
Climatic Integration for Spoilage:
SpoilageEngine fetches climate data. It does this without directly passing coordinates for items stored in inventories by looking up the BlockPos of the Container block entity via Level. If an item is dropped on the ground in the Permafrost biome (HorizonWorld / HorizonBiome), the localized ClimateSnapshot ambient temperature effectively halts the decay timer naturally.
Disease Deficiencies Bridge:
The DigestionEngine checks for nutrient deficits below a strict 5.0f threshold. When a threshold is broken, rather than handling the debuffs itself, it fires a NutrientDeficiencyEvent. HorizonBiome listens for this event to apply actual DiseaseType active infections (e.g., Scurvy, Rickets), keeping the pathogenic and nutritional systems cleanly decoupled.
Cross-Mod Medical Compatibility:
MedicalTreatment entries define required skills as Strings (e.g., "medicine", "science"). These strings map directly to SkillType.name() stored inside HorizonCore's player capability. If HorizonSkills is installed, these checks become much more granular, utilizing dynamic RPG levels instead of base integers.
Bloodletting Historical Accuracy Limit:
The bloodletting treatment is explicitly designed to hurt the player (-4.0f heal amount). However, it is registered under the MedicalTreatment schema. NPC civilizations in HorizonSociety running Iron Age or Medieval government AIs will actively attempt to administer this "cure" to player characters who suffer from unrelated diseases, making early-age NPC hospitals actively dangerous.
