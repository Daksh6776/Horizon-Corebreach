1. Overview & Responsibilities
HorizonKnowledge simulates the discovery, processing, and transmission of ideas, historical records, and moral frameworks throughout civilization.
Archaeology & Antiquity: Players and NPCs excavate ancient ruins. Artifacts are unearthed, carbon-dated, and placed in Museums to generate cultural prestige and reverse-engineer lost technologies.
Philosophy & Ethics: 10 distinct philosophical schools compete for dominance in the minds of the population, shaping a civilization's moral compass. This governs how NPCs react to player actions (e.g., Utilitarian societies accept forced labor if it maximizes overall output, while Existentialist societies revolt).
Communication Networks: Information does not travel instantly. The spread of technology, diplomatic incidents, and cultural shifts is gated by the communication infrastructure, ranging from foot messengers to quantum entanglement.
Knowledge Infrastructure: 8 distinct building types that act as nodes for generating research, storing artifacts, and amplifying communication ranges.

package com.horizonpack.horizonknowledge.systems;

import com.horizonpack.horizonknowledge.data.components.ArtifactData;
import com.horizonpack.horizonknowledge.data.components.KnowledgeComponents;
import com.horizonpack.horizonknowledge.items.ArtifactItemRegistry;
import net.minecraft.core.BlockPos;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.world.entity.player.Player;
import net.minecraft.world.item.ItemStack;

import java.util.Random;

public class ArchaeologyEngine {

    public static void init() {}

    public static void performExcavation(ServerLevel level, BlockPos pos, Player player) {
        Random rand = level.getRandom();
        
        // 10% chance to find an artifact when brushing suspicious blocks
        if (rand.nextFloat() < 0.10f) {
            ItemStack artifactStack = generateRandomArtifact(level, rand);
            
            // Assign carbon dating based on world coordinates and depth
            int estimatedAge = (64 - pos.getY()) * 100 + rand.nextInt(500);
            artifactStack.set(KnowledgeComponents.ARTIFACT_DATA.get(), new ArtifactData(estimatedAge, player.getUUID(), pos));
            
            net.minecraft.world.entity.item.ItemEntity entity = new net.minecraft.world.entity.item.ItemEntity(level, pos.getX() + 0.5, pos.getY() + 1.0, pos.getZ() + 0.5, artifactStack);
            level.addFreshEntity(entity);
            
            // Fire event for advancements/skills
            net.neoforged.neoforge.common.NeoForge.EVENT_BUS.post(new com.horizonpack.horizonknowledge.events.custom.ArtifactExcavatedEvent(player, artifactStack, pos));
        }
    }

    private static ItemStack generateRandomArtifact(ServerLevel level, Random rand) {
        float roll = rand.nextFloat();
        if (roll < 0.01f) return new ItemStack(ArtifactItemRegistry.ALIEN_ARTIFACT.get());
        if (roll < 0.05f) return new ItemStack(ArtifactItemRegistry.LOST_TECHNOLOGY_BLUEPRINT.get());
        if (roll < 0.15f) return new ItemStack(ArtifactItemRegistry.MEDIEVAL_MANUSCRIPT.get());
        if (roll < 0.30f) return new ItemStack(ArtifactItemRegistry.ROMAN_COIN.get());
        if (roll < 0.45f) return new ItemStack(ArtifactItemRegistry.ANCIENT_SCROLL.get());
        if (roll < 0.60f) return new ItemStack(ArtifactItemRegistry.IRON_RELIC.get());
        if (roll < 0.75f) return new ItemStack(ArtifactItemRegistry.BRONZE_FIGURINE.get());
        if (roll < 0.90f) return new ItemStack(ArtifactItemRegistry.CLAY_TABLET.get());
        return new ItemStack(ArtifactItemRegistry.FLINT_TOOL.get());
    }
}

package com.horizonpack.horizonknowledge.systems;

import com.horizonpack.horizonknowledge.data.KnowledgeSavedData;
import com.horizonpack.horizonsociety.state.Civilization;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.server.level.ServerLevel;

import java.util.UUID;

public class CommunicationEngine {

    public static void init() {}

    /**
     * Determines how long (in ticks) it takes for a technology or diplomatic event to reach a target civilization.
     */
    public static int calculateTransmissionDelay(ServerLevel level, Civilization source, Civilization target) {
        KnowledgeSavedData data = KnowledgeSavedData.get(level);
        
        ResourceLocation sourceNetId = data.getHighestNetwork(source.getId());
        ResourceLocation targetNetId = data.getHighestNetwork(target.getId());
        
        float sourceSpeed = getSpeedModifier(level, sourceNetId);
        float targetSpeed = getSpeedModifier(level, targetNetId);
        
        // Base delay is 1 in-game week (168,000 ticks)
        int baseDelay = 168000;
        
        // The highest communication network between the two dictates the speed
        float effectiveSpeed = Math.max(sourceSpeed, targetSpeed);
        
        return (int) (baseDelay / effectiveSpeed);
    }
    
    private static float getSpeedModifier(ServerLevel level, ResourceLocation networkId) {
        var network = level.registryAccess().registryOrThrow(com.horizonpack.horizonknowledge.core.KnowledgeRegistries.COMM_NETWORK_KEY).get(networkId);
        return network != null ? network.transmissionSpeedMultiplier() : 0.1f;
    }
}

    // Civ UUID -> Highest communication network ID
    private final Map<UUID, ResourceLocation> civNetworks = new HashMap<>();

    public static KnowledgeSavedData get(ServerLevel level) {
        return level.getDataStorage().computeIfAbsent(
                new SavedData.Factory<>(KnowledgeSavedData::new, KnowledgeSavedData::load),
                DATA_NAME
        );
    }

    public void shiftPlayerAlignment(UUID playerId, ResourceLocation philosophy, float weight) {
        Map<ResourceLocation, Float> alignments = playerAlignments.computeIfAbsent(playerId, k -> new HashMap<>());
        alignments.put(philosophy, alignments.getOrDefault(philosophy, 0.0f) + weight);
        setDirty();
    }

    public ResourceLocation getDominantPhilosophy(UUID civId) {
        // Return dominant philosophy based on regional calculations (simplified)
        return ResourceLocation.fromNamespaceAndPath("horizonknowledge", "pragmatism");
    }

    public ResourceLocation getHighestNetwork(UUID civId) {
        return civNetworks.getOrDefault(civId, ResourceLocation.fromNamespaceAndPath("horizonknowledge", "messenger_system"));
    }
    
    public void upgradeNetwork(UUID civId, ResourceLocation newNetwork) {
        civNetworks.put(civId, newNetwork);
        setDirty();
    }

    @Override
    public @NotNull CompoundTag save(@NotNull CompoundTag tag) {
        ListTag playersTag = new ListTag();
        playerAlignments.forEach((uuid, map) -> {
            CompoundTag pTag = new CompoundTag();
            pTag.putUUID("UUID", uuid);
            CompoundTag alignments = new CompoundTag();
            map.forEach((phil, weight) -> alignments.putFloat(phil.toString(), weight));
            pTag.put("Alignments", alignments);
            playersTag.add(pTag);
        });
        tag.put("PlayerAlignments", playersTag);
        
        CompoundTag networksTag = new CompoundTag();
        civNetworks.forEach((uuid, net) -> networksTag.putString(uuid.toString(), net.toString()));
        tag.put("CivNetworks", networksTag);
        
        return tag;
    }

    public static KnowledgeSavedData load(CompoundTag tag) {
        KnowledgeSavedData data = new KnowledgeSavedData();
        ListTag playersTag = tag.getList("PlayerAlignments", Tag.TAG_COMPOUND);
        for (int i = 0; i < playersTag.size(); i++) {
            CompoundTag pTag = playersTag.getCompound(i);
            UUID uuid = pTag.getUUID("UUID");
            CompoundTag alignments = pTag.getCompound("Alignments");
            Map<ResourceLocation, Float> map = new HashMap<>();
            for (String key : alignments.getAllKeys()) {
                map.put(ResourceLocation.parse(key), alignments.getFloat(key));
            }
            data.playerAlignments.put(uuid, map);
        }
        
        CompoundTag networksTag = tag.getCompound("CivNetworks");
        for (String key : networksTag.getAllKeys()) {
            data.civNetworks.put(UUID.fromString(key), ResourceLocation.parse(networksTag.getString(key)));
        }
        
        return data;
    }
}

26. Critical Implementation Implementation Notes
Artifact Data Component Validation: The ArtifactData component stores the absolute BlockPos and UUID of the discoverer. To prevent exploiters from spoofing network packets to forge artifact authenticity (inflating museum prestige), the component is strictly immutable. If a player attempts to merge artifacts in an anvil, the output strips the ArtifactData component entirely.
Museum Prestige Ticking Strategy: The MuseumBlockEntity evaluates its inventory every 24,000 ticks. To avoid chunk-unload synchronization bugs, it does not directly query horizonsociety settlements. Instead, it adds to an internal accumulatedPrestige float. When a settlement's daily AI Director scan passes over the chunk, the settlement pulls the prestige directly and clears the buffer.
Communication Network Calculations: CommunicationEngine.calculateTransmissionDelay() handles information lag between Civilizations. Because distances can span millions of blocks, the math uses Math.max(sourceSpeed, targetSpeed). If both networks are messenger_system, the delay is purely Euclidean block distance divided by base human speed.
Data Pack Extensibility of Philosophies: PhilosophicalSchool utilizes a ResourceKey<Registry<T>> built with .sync(true). Modpack developers can add entirely custom ethical frameworks (e.g., Objectivism) by simply dropping a new JSON into the philosophical_schools datapack folder. The HUD overlay will automatically populate it.
Ethics Inter-Mod Routing: PhilosophyEngine.processMoralAction() acts as a receiver for events fired globally by horizoncore and horizonsociety. Do not hard-reference vanilla Player interactions (like animal killing) here directly; rely on custom tags and MoralDecisionEvent dispatches to maintain clean boundaries between modules.
Carbon Dating NBT Math: ArtifactData.estimatedAgeYears generates an age based on the depth (Y-level) at which it was excavated, scaled by the chunk's HorizonWorld geological profile. This guarantees artifacts excavated from deep shale layers reliably date older than those in surface loam.
Supercomputer Energy Drain Integration: The SupercomputerLabBlockEntity requires horizonpower. Use level.getCapability(Capabilities.EnergyStorage.BLOCK, pos, null) to check for available RF. If power drops to 0, the machine's state boolean flips, severing its communication network boost.
Player Ethics Serialization: Player alignments to various philosophies are saved within KnowledgeSavedData rather than IHorizonPlayerData to allow the Civilization AI Director to scan regional moral trends quickly without having to load offline player capability files.
