1. Overview & Responsibilities
HorizonCulture implements the non-material evolution of civilization, transforming mere survival into a living society. It governs the flow of ideas, art, faith, and communication.
Religion: Tracks faith points, divine favor, schisms, and holy wars. Drives NPC civilization happiness and geopolitical friction.
Linguistics: A dynamic language simulation. Chat and written texts are obfuscated based on a player’s exact proficiency (0.0 to 1.0) in the specific language family.
Literature & Writing: Physical transmission of knowledge. Books evolve from clay tablets to mass-produced newspapers, acting as tech-boosters and propaganda tools.
Music & Art: Usable instruments that grant localized buffs, and craftable art pieces that boost settlement prestige.

package com.horizonpack.horizonculture.items;

import net.minecraft.core.BlockPos;
import net.minecraft.sounds.SoundEvent;
import net.minecraft.sounds.SoundSource;
import net.minecraft.world.InteractionHand;
import net.minecraft.world.InteractionResultHolder;
import net.minecraft.world.entity.player.Player;
import net.minecraft.world.item.Item;
import net.minecraft.world.item.ItemStack;
import net.minecraft.world.level.Level;
import java.util.function.Supplier;

public class InstrumentItem extends Item {
    private final Supplier<SoundEvent> instrumentSound;
    private final int buffRadius;
    private final String instrumentType;

    public InstrumentItem(Properties properties, Supplier<SoundEvent> sound, int radius, String type) {
        super(properties);
        this.instrumentSound = sound;
        this.buffRadius = radius;
        this.instrumentType = type;
    }

    @Override
    public InteractionResultHolder<ItemStack> use(Level level, Player player, InteractionHand hand) {
        ItemStack stack = player.getItemInHand(hand);
        level.playSound(null, player.getX(), player.getY(), player.getZ(), instrumentSound.get(), SoundSource.RECORDS, 1.0f, 1.0f);
        
        if (!level.isClientSide) {
            // Apply localized morale/culture buff via CultureEngine
            com.horizonpack.horizonculture.systems.CultureEngine.applyMusicalBuff(level, player.blockPosition(), buffRadius, instrumentType);
            player.getCooldowns().addCooldown(this, 100); // 5 second cooldown
        }
        return InteractionResultHolder.sidedSuccess(stack, level.isClientSide());
    }
}

package com.horizonpack.horizonculture.utils;

import net.minecraft.network.chat.Component;
import net.minecraft.network.chat.MutableComponent;
import net.minecraft.network.chat.Style;

import java.util.Random;

public class ObfuscationMath {

    /**
     * Translates a string based on the listener's proficiency (0.0 = completely scrambled, 1.0 = clear).
     */
    public static Component obfuscateText(String sourceText, float proficiency, long randomSeed) {
        if (proficiency >= 0.95f) return Component.literal(sourceText);

        Random rand = new Random(randomSeed);
        StringBuilder output = new StringBuilder();
        
        for (char c : sourceText.toCharArray()) {
            if (Character.isWhitespace(c)) {
                output.append(c);
            } else if (rand.nextFloat() > proficiency) {
                // Scramble Character
                char scrambled = (char) ('a' + rand.nextInt(26));
                output.append(scrambled);
            } else {
                output.append(c);
            }
        }
        
        MutableComponent comp = Component.literal(output.toString());
        if (proficiency < 0.3f) {
            comp.setStyle(Style.EMPTY.withObfuscated(true).withColor(0x888888));
        }
        return comp;
    }
}

package com.horizonpack.horizonculture.systems;

import com.horizonpack.horizoncore.capabilities.HorizonCapabilities;
import com.horizonpack.horizoncore.capabilities.IHorizonPlayerData;
import com.horizonpack.horizonculture.data.CultureSavedData;
import net.minecraft.core.BlockPos;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.world.entity.player.Player;
import net.minecraft.world.phys.AABB;

import java.util.List;

public class CultureEngine {

    public static void init() {}

    public static void tickCulture(ServerLevel level) {
        CultureSavedData data = CultureSavedData.get(level);
        
        // Background tick: expand religious borders and decay unused languages
        // Called every DayAdvanceEvent from HorizonCore.
        data.tickInfluenceMap();
    }

    public static void applyMusicalBuff(ServerLevel level, BlockPos source, int radius, String type) {
        AABB bounds = new AABB(source).inflate(radius);
        List<Player> players = level.getEntitiesOfClass(Player.class, bounds);
        
        for (Player p : players) {
            // Apply buffs based on instrument type. Primitive drums grant speed/strength.
            if (type.equals("primitive")) {
                p.addEffect(new net.minecraft.world.effect.MobEffectInstance(net.minecraft.world.effect.MobEffects.MOVEMENT_SPEED, 200, 0));
                p.addEffect(new net.minecraft.world.effect.MobEffectInstance(net.minecraft.world.effect.MobEffects.DAMAGE_BOOST, 200, 0));
            } else if (type.equals("classical")) {
                p.addEffect(new net.minecraft.world.effect.MobEffectInstance(net.minecraft.world.effect.MobEffects.REGENERATION, 200, 0));
            }
        }
    }
    
    public static void readWrittenWork(Player player, String languageId, float knowledgeValue) {
        IHorizonPlayerData data = player.getCapability(HorizonCapabilities.PLAYER_DATA).orElse(null);
        if (data != null) {
            float prof = data.getLanguageProficiency(languageId);
            if (prof < 1.0f) {
                // Reading books slowly increases language proficiency
                data.improveLanguage(languageId, 0.05f);
            } else {
                // If fluent, applies the knowledge/research directly to the tech tree
                // com.horizonpack.horizoncore.systems.ResearchEngine.addBonus(knowledgeValue);
            }
        }
    }
}

package com.horizonpack.horizonculture.data;

import java.util.HashMap;
import java.util.Map;
import java.util.UUID;

public class CultureSavedData extends SavedData {

    public static final String DATA_NAME = "horizonculture_state";

    // Civilization UUID -> Religion ID
    private final Map<UUID, ResourceLocation> civReligions = new HashMap<>();
    
    // Civilization UUID -> Language ID
    private final Map<UUID, ResourceLocation> civLanguages = new HashMap<>();

    public static CultureSavedData get(ServerLevel level) {
        return level.getDataStorage().computeIfAbsent(
                new SavedData.Factory<>(CultureSavedData::new, CultureSavedData::load),
                DATA_NAME
        );
    }

    public ResourceLocation getCivReligion(UUID civId) {
        return civReligions.getOrDefault(civId, ResourceLocation.fromNamespaceAndPath("horizonculture", "animism"));
    }

    public void setCivReligion(UUID civId, ResourceLocation religion) {
        civReligions.put(civId, religion);
        setDirty();
    }

    public void tickInfluenceMap() {
        // Border expansion mathematics for religious pressure
        setDirty();
    }

    @Override
    public @NotNull CompoundTag save(@NotNull CompoundTag tag) {
        ListTag relTag = new ListTag();
        civReligions.forEach((uuid, rel) -> {
            CompoundTag c = new CompoundTag();
            c.putUUID("Civ", uuid);
            c.putString("Rel", rel.toString());
            relTag.add(c);
        });
        tag.put("Religions", relTag);
        return tag;
    }

    public static CultureSavedData load(CompoundTag tag) {
        CultureSavedData data = new CultureSavedData();
        ListTag relTag = tag.getList("Religions", Tag.TAG_COMPOUND);
        for (int i = 0; i < relTag.size(); i++) {
            CompoundTag c = relTag.getCompound(i);
            data.civReligions.put(c.getUUID("Civ"), ResourceLocation.parse(c.getString("Rel")));
        }
        return data;
    }
}

import org.jetbrains.annotations.Nullable;

public class PrintingPressBlockEntity extends BlockEntity implements MenuProvider {

    // 0: Master Copy, 1: Paper Input, 2: Ink Input, 3: Output
    public final ItemStackHandler inventory = new ItemStackHandler(4) {
        @Override
        protected void onContentsChanged(int slot) {
            setChanged();
        }
    };

    private int progress = 0;
    private static final int MAX_PROGRESS = 100;

    public PrintingPressBlockEntity(BlockPos pos, BlockState state) {
        super(CultureRegistries.PRINTING_PRESS_BE.get(), pos, state);
    }

    public static void serverTick(Level level, BlockPos pos, BlockState state, PrintingPressBlockEntity be) {
        if (!be.inventory.getStackInSlot(0).isEmpty() && !be.inventory.getStackInSlot(1).isEmpty() && !be.inventory.getStackInSlot(2).isEmpty()) {
            if (be.inventory.getStackInSlot(3).getCount() < 64) {
                be.progress++;
                if (be.progress >= MAX_PROGRESS) {
                    be.inventory.getStackInSlot(1).shrink(1); // Consume Paper
                    be.inventory.getStackInSlot(2).shrink(1); // Consume Ink
                    
                    ItemStack output = be.inventory.getStackInSlot(0).copy();
                    output.setCount(1);
                    
                    if (be.inventory.getStackInSlot(3).isEmpty()) {
                        be.inventory.setStackInSlot(3, output);
                    } else {
                        be.inventory.getStackInSlot(3).grow(1);
                    }
                    
                    be.progress = 0;
                    be.setChanged();
                }
                return;
            }
        }
        if (be.progress > 0) {
            be.progress--;
            be.setChanged();
        }
    }

    @Override
    protected void saveAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.saveAdditional(tag, registries);
        tag.put("Inventory", inventory.serializeNBT(registries));
        tag.putInt("Progress", progress);
    }

    @Override
    protected void loadAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.loadAdditional(tag, registries);
        if (tag.contains("Inventory")) inventory.deserializeNBT(registries, tag.getCompound("Inventory"));
        progress = tag.getInt("Progress");
    }

    @Override
    public Component getDisplayName() {
        return Component.translatable("block.horizonculture.printing_press");
    }

    @Nullable
    @Override
    public AbstractContainerMenu createMenu(int containerId, Inventory inventory, Player player) {
        return null; // Linked to GUI container in implementation
    }

25. Critical Implementation Notes
Client-Side Obfuscation Injection:
Modifying chat directly via ServerChatEvent is unsafe for formatting compatibility with other mods. Instead, the server must calculate proficiency and dispatch the TranslateMessagePacket. The client intercepts this in a localized GUI render hook (e.g., ClientChatReceivedEvent) and applies the ObfuscationMath algorithm locally to retain format integrity while scrambling the payload based on synced capability data.
Language Font Rendering:
To accurately portray distinct language roots (e.g., Mesoamerican vs Cyrillic), horizonculture dynamically injects resource pack font mappings. When obfuscation fires for an unknown language, it replaces standard A-Z characters with Unicode glyphs assigned to a custom TTF font loaded in the mod's assets/minecraft/font/default.json overrides.
Printing Press Performance:
The PrintingPressBlockEntity performs a deep copy of complex WrittenBookItem NBT components. To prevent server lag spikes when copying massive 50-page books containing hundreds of JSON text components, the deep copy operation in serverTick must be segmented, or restricted to cloning the fundamental Data Component hash map rather than reparsing the strings.
Instrument Sound Attenuation:
By default, Minecraft OpenAL culls sounds beyond 16 blocks. Because instruments like the Church Organ or Drum need to be heard across an entire settlement, the registered SoundEvent must use SoundEvent.createFixedRangeEvent(id, range) with a radius set to 64 or 128 in CultureRegistries.java.
Musical Instrument MIDI Parsing:
Advanced instruments (Piano, Synthesizer) do not just play one static sound. They utilize a custom MidiPlaybackHelper that reads simple byte-arrays stored in NBT (from "Sheet Music" items) and sequentially plays individual pitch-shifted note particles over several seconds using a localized tick queue.
Dynamic Art Canvas NBT Limits:
ArtCanvasBlockEntity allows players to "paint" by saving 16x16 pixel grids into the BlockEntity NBT. A 256-int array can quickly exceed network packet limits if a player builds a gallery of 100 canvases. Ensure getUpdatePacket() utilizes a tightly packed byte[] format representing a custom indexed color palette instead of raw RGB ints.
Religion Influence Mathematics:
CultureSavedData.tickInfluenceMap() evaluates religious pressure using a Voronoi diagram approximation. To prevent total server lockup when scanning thousands of settlement chunks, the data structure stores only "Epicenter" coordinates and applies a basic inverse-square distance calculation upon chunk load, rather than iterating every block coordinate globally.
Holy War & Horizonsociety Integration:
ReligionType contains the allowsHolyWar boolean. When CultureEngine detects border friction between two civilizations with overlapping religious pressure, it injects a hostile diplomatic modifier directly into horizonsociety's DiplomacyEngine. If both civs have allowsHolyWar = true, the aggression multiplier squares itself, bypassing standard trade-pact restrictions.
