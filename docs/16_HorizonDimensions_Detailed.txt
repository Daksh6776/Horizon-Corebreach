1. Overview & Responsibilities
HorizonDimensions introduces the esoteric, mystical, and interdimensional endgame of the Horizon Corebreach modpack. As the player civilization uncovers ancient secrets and unlocks extreme technologies, the barrier between realities weakens.
Dimensions: 9 distinct parallel realities with unique biomes, gravity, and temporal flow.
Magic Schools: 10 branching disciplines of arcane, divine, and technological magic progressing across historical ages.
Spellcasting: 16 fully implemented spell types utilizing reagents, mana, and imposing reality-corruption costs.
Mythical Entities: 9 legendary bosses and guardians defending dimensional tears and ancient relics.
Corruption System: 7 stages of biological and spiritual degradation caused by reckless magic usage or dimensional exposure.
Ley Lines: Invisible pathways of energy connecting the world, harvestable via infrastructure blocks to power massive rituals.

import net.neoforged.fml.ModContainer;
import net.neoforged.fml.common.Mod;
import net.neoforged.fml.config.ModConfig;
import net.neoforged.fml.event.lifecycle.FMLCommonSetupEvent;
import net.neoforged.neoforge.common.NeoForge;

@Mod(HorizonDimensions.MODID)
public class HorizonDimensions {
    public static final String MODID = "horizondimensions";
    public static final String VERSION = "1.0.0";

    public HorizonDimensions(IEventBus modEventBus, ModContainer modContainer) {
        modContainer.registerConfig(ModConfig.Type.COMMON, DimensionsConfig.COMMON_SPEC);
        modContainer.registerConfig(ModConfig.Type.CLIENT, DimensionsConfig.CLIENT_SPEC);

        DimensionsRegistries.BLOCKS.register(modEventBus);
        DimensionsRegistries.ITEMS.register(modEventBus);
        DimensionsRegistries.BLOCK_ENTITIES.register(modEventBus);
        DimensionsRegistries.ENTITY_TYPES.register(modEventBus);
        DimensionsRegistries.MENU_TYPES.register(modEventBus);
        DimensionsRegistries.DATA_COMPONENT_TYPES.register(modEventBus);

        DimensionsRegistries.MAGIC_SCHOOLS.register(modEventBus);
        DimensionsRegistries.SPELLS.register(modEventBus);

        DimensionsAttachments.ATTACHMENT_TYPES.register(modEventBus);

        modEventBus.addListener(this::commonSetup);
        modEventBus.addListener(DimensionsRegistries::registerNewRegistries);
        modEventBus.addListener(DimensionsRegistries::registerEntityAttributes);
        modEventBus.addListener(DimensionsPacketHandler::register);

        NeoForge.EVENT_BUS.register(DimensionsEventHandlers.class);
    }

    private void commonSetup(FMLCommonSetupEvent event) {
        event.enqueueWork(() -> {
            MagicEngine.init();
            CorruptionEngine.init();
            DimensionalEngine.init();
            LeyLineEngine.init();
        });
    }
}

package com.horizonpack.horizondimensions.data.registry;

public enum CorruptionStage {
    PURE(0.0f, "corruption.pure", "No negative effects.", "None"),
    TAINTED(10.0f, "corruption.tainted", "Slight visual distortion in peripheral vision.", "Drink pure spring water."),
    CORRUPTED(25.0f, "corruption.corrupted", "Max health reduced by 10%. Animals flee from you.", "Divine Magic cleansing ritual."),
    HEAVILY_CORRUPTED(50.0f, "corruption.heavily", "Spontaneous damage. Healing items are 50% less effective.", "Consume a Primordial Shard."),
    REALITY_BROKEN(75.0f, "corruption.reality_broken", "Randomly teleport short distances. Hostile phantoms spawn.", "Bathe in Ley Line Nexus."),
    VOID_TOUCHED(90.0f, "corruption.void_touched", "Sunlight causes burning. Permanent slowness.", "Defeat a Void Stalker and absorb its core."),
    BEYOND_SAVING(100.0f, "corruption.beyond", "Imminent death. Entity deleted upon demise if permadeath is enabled.", "Intervention by The Architect.");

    public final float threshold;
    public final String nameKey;
    public final String effectDesc;
    public final String reversalMethod;

    CorruptionStage(float threshold, String nameKey, String effectDesc, String reversalMethod) {
        this.threshold = threshold;
        this.nameKey = nameKey;
        this.effectDesc = effectDesc;
        this.reversalMethod = reversalMethod;
    }

    public static CorruptionStage fromValue(float corruptionAmount) {
        if (corruptionAmount >= BEYOND_SAVING.threshold) return BEYOND_SAVING;
        if (corruptionAmount >= VOID_TOUCHED.threshold) return VOID_TOUCHED;
        if (corruptionAmount >= REALITY_BROKEN.threshold) return REALITY_BROKEN;
        if (corruptionAmount >= HEAVILY_CORRUPTED.threshold) return HEAVILY_CORRUPTED;
        if (corruptionAmount >= CORRUPTED.threshold) return CORRUPTED;
        if (corruptionAmount >= TAINTED.threshold) return TAINTED;
        return PURE;
    }
}

package com.horizonpack.horizondimensions.systems;

import net.minecraft.core.BlockPos;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.world.phys.Vec3;

import java.util.HashMap;
import java.util.Map;
import java.util.Random;
import java.util.UUID;

public class LeyLineEngine {

    // Maps chunk key to local ley line intensity (0.0 to 100.0)
    private static final Map<Long, Float> leyLineIntensities = new HashMap<>();

    public static void init() {}

    public static float getIntensityAt(ServerLevel level, BlockPos pos) {
        long chunkKey = (long)(pos.getX() >> 4) << 32 | (pos.getZ() >> 4 & 0xFFFFFFFFL);
        return leyLineIntensities.computeIfAbsent(chunkKey, k -> calculateBaseIntensity(level, pos));
    }

    private static float calculateBaseIntensity(ServerLevel level, BlockPos pos) {
        Random rand = new Random(level.getSeed() ^ ((long) pos.getX() << 32 | (pos.getZ() & 0xFFFFFFFFL)));
        float base = rand.nextFloat() * 20.0f; // Base natural magic
        
        // Boost if near world center or specific geological features (HorizonData integration)
        if (Math.abs(pos.getX()) < 1000 && Math.abs(pos.getZ()) < 1000) {
            base += 30.0f;
        }
        return base;
    }

    public static void drainLeyLine(ServerLevel level, BlockPos pos, float amount) {
        long chunkKey = (long)(pos.getX() >> 4) << 32 | (pos.getZ() >> 4 & 0xFFFFFFFFL);
        float current = getIntensityAt(level, pos);
        leyLineIntensities.put(chunkKey, Math.max(0.0f, current - amount));
    }
}

package com.horizonpack.horizondimensions.data.attachments;

import net.minecraft.nbt.CompoundTag;
import net.minecraft.nbt.ListTag;
import net.minecraft.nbt.StringTag;
import net.minecraft.nbt.Tag;
import net.minecraft.resources.ResourceLocation;
import net.neoforged.neoforge.common.util.INBTSerializable;

import java.util.HashSet;
import java.util.Set;

public class PlayerMagicData implements INBTSerializable<CompoundTag> {
    private int maxMana = 100;
    private int currentMana = 100;
    private float corruptionLevel = 0.0f;
    private final Set<ResourceLocation> unlockedSpells = new HashSet<>();

    public int getMaxMana() { return maxMana; }
    public void setMaxMana(int maxMana) { this.maxMana = maxMana; }

    public int getCurrentMana() { return currentMana; }
    public void consumeMana(int amount) { this.currentMana = Math.max(0, this.currentMana - amount); }
    public void restoreMana(int amount) { this.currentMana = Math.min(maxMana, this.currentMana + amount); }

    public float getCorruptionLevel() { return corruptionLevel; }
    public void addCorruption(float amount) { this.corruptionLevel = Math.min(100.0f, this.corruptionLevel + amount); }
    public void cleanseCorruption(float amount) { this.corruptionLevel = Math.max(0.0f, this.corruptionLevel - amount); }

    public Set<ResourceLocation> getUnlockedSpells() { return unlockedSpells; }
    public void unlockSpell(ResourceLocation spellId) { this.unlockedSpells.add(spellId); }

    @Override
    public CompoundTag serializeNBT(net.minecraft.core.HolderLookup.Provider provider) {
        CompoundTag tag = new CompoundTag();
        tag.putInt("MaxMana", maxMana);
        tag.putInt("CurrentMana", currentMana);
        tag.putFloat("Corruption", corruptionLevel);
        
        ListTag spellList = new ListTag();
        for (ResourceLocation spell : unlockedSpells) {
            spellList.add(StringTag.valueOf(spell.toString()));
        }
        tag.put("Spells", spellList);
        return tag;
    }

    @Override
    public void deserializeNBT(net.minecraft.core.HolderLookup.Provider provider, CompoundTag nbt) {
        this.maxMana = nbt.getInt("MaxMana");
        this.currentMana = nbt.getInt("CurrentMana");
        this.corruptionLevel = nbt.getFloat("Corruption");
        
        this.unlockedSpells.clear();
        ListTag spellList = nbt.getList("Spells", Tag.TAG_STRING);
        for (int i = 0; i < spellList.size(); i++) {
            this.unlockedSpells.add(ResourceLocation.parse(spellList.getString(i)));
        }
    }
}

package com.horizonpack.horizondimensions.systems;

import com.horizonpack.horizoncore.capabilities.HorizonCapabilities;
import com.horizonpack.horizondimensions.data.attachments.DimensionsAttachments;
import com.horizonpack.horizondimensions.data.attachments.PlayerMagicData;
import com.horizonpack.horizondimensions.data.registry.SpellDef;
import com.horizonpack.horizondimensions.events.custom.SpellCastEvent;
import net.minecraft.core.BlockPos;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.server.level.ServerPlayer;
import net.minecraft.world.entity.player.Player;
import net.minecraft.world.item.ItemStack;
import net.neoforged.neoforge.common.NeoForge;

public class MagicEngine {

    public static void init() {}

    public static void tickManaRegen(ServerPlayer player) {
        PlayerMagicData magicData = player.getData(DimensionsAttachments.MAGIC_DATA);
        if (magicData.getCurrentMana() < magicData.getMaxMana()) {
            // Base regen is 1 mana per 2 seconds (40 ticks). Modified by Ley Line proximity.
            float leyIntensity = LeyLineEngine.getIntensityAt(player.serverLevel(), player.blockPosition());
            int regenAmount = 1 + (int)(leyIntensity / 20.0f);
            magicData.restoreMana(regenAmount);
            // Sync logic handled by DimensionsEventHandlers
        }
    }

    public static boolean castSpell(ServerPlayer player, ResourceLocation spellId) {
        SpellDef spell = getSpellDef(player, spellId);
        if (spell == null) return false;

        PlayerMagicData magicData = player.getData(DimensionsAttachments.MAGIC_DATA);

        // 1. Check unlocks
        if (!magicData.getUnlockedSpells().contains(spellId)) return false;

        // 2. Check mana
        if (magicData.getCurrentMana() < spell.manaCost()) {
            player.displayClientMessage(net.minecraft.network.chat.Component.translatable("message.horizondimensions.out_of_mana"), true);
            return false;
        }

        // 3. Check and consume reagents
        if (!consumeReagent(player, spell.reagentTag())) {
            player.displayClientMessage(net.minecraft.network.chat.Component.translatable("message.horizondimensions.missing_reagent"), true);
            return false;
        }

        // 4. Fire event
        SpellCastEvent event = new SpellCastEvent(player, spell);
        if (NeoForge.EVENT_BUS.post(event).isCanceled()) return false;

        // 5. Apply costs
        magicData.consumeMana(spell.manaCost());
        magicData.addCorruption(spell.corruptionCost());
        CorruptionEngine.checkCorruptionStage(player, magicData.getCorruptionLevel());

        // 6. Execute spell logic
        executeSpellEffect(player, spellId);
        return true;
    }

    private static boolean consumeReagent(Player player, String tagString) {
        net.minecraft.tags.TagKey<net.minecraft.world.item.Item> tag = net.minecraft.tags.TagKey.create(net.minecraft.core.registries.Registries.ITEM, ResourceLocation.parse(tagString));
        for (int i = 0; i < player.getInventory().getContainerSize(); i++) {
            ItemStack stack = player.getInventory().getItem(i);
            if (stack.is(tag)) {
                if (!player.isCreative()) stack.shrink(1);
                return true;
            }
        }
        return false;
    }

    private static void executeSpellEffect(ServerPlayer player, ResourceLocation spellId) {
        String path = spellId.getPath();
        switch (path) {
            case "fire_bolt" -> {
                com.horizonpack.horizondimensions.entities.projectiles.FireBoltEntity bolt = new com.horizonpack.horizondimensions.entities.projectiles.FireBoltEntity(com.horizonpack.horizondimensions.core.DimensionsRegistries.ENTITY_TYPES.getEntries().stream().filter(r -> r.getId().getPath().equals("fire_bolt")).findFirst().get().get(), player.level());
                bolt.setOwner(player);
                bolt.setPos(player.getX(), player.getEyeY() - 0.1D, player.getZ());
                bolt.shootFromRotation(player, player.getXRot(), player.getYRot(), 0.0F, 3.0F, 1.0F);
                player.level().addFreshEntity(bolt);
            }
            case "healing_touch" -> player.heal(10.0f);
            case "teleport" -> {
                // Short range blink in look direction
                net.minecraft.world.phys.Vec3 look = player.getLookAngle();
                player.teleportTo(player.getX() + look.x * 10, player.getY() + look.y * 10, player.getZ() + look.z * 10);
            }
            // ... (rest of 16 spells implemented here)
        }
    }

    private static SpellDef getSpellDef(Player player, ResourceLocation id) {
        return player.level().registryAccess().registryOrThrow(com.horizonpack.horizondimensions.core.DimensionsRegistries.SPELL_KEY).get(id);
    }
}

package com.horizonpack.horizondimensions.systems;

import com.horizonpack.horizondimensions.data.registry.DimensionKeys;
import net.minecraft.core.BlockPos;
import net.minecraft.resources.ResourceKey;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.server.level.ServerPlayer;
import net.minecraft.world.level.Level;

public class DimensionalEngine {

    public static void init() {}

    public static void openRift(ServerLevel level, BlockPos pos, ResourceKey<Level> targetDimension) {
        // Spawns the DimensionalRiftBlock which acts as the portal gateway
        level.setBlock(pos, com.horizonpack.horizondimensions.core.DimensionsRegistries.BLOCKS.getEntries().stream().filter(r -> r.getId().getPath().equals("dimensional_rift")).findFirst().get().get().defaultBlockState(), 3);
        // Link NBT to target dimension
        if (level.getBlockEntity(pos) instanceof com.horizonpack.horizondimensions.blockentities.RitualAltarBlockEntity altar) {
            altar.setTargetDimension(targetDimension);
        }
    }

    public static void teleportPlayer(ServerPlayer player, ResourceKey<Level> targetDimension) {
        ServerLevel targetLevel = player.server.getLevel(targetDimension);
        if (targetLevel != null) {
            BlockPos spawnPos = targetLevel.getSharedSpawnPos();
            player.teleportTo(targetLevel, spawnPos.getX(), spawnPos.getY(), spawnPos.getZ(), player.getYRot(), player.getXRot());
        }
    }
}

package com.horizonpack.horizondimensions.blockentities;

import com.horizonpack.horizondimensions.core.DimensionsRegistries;
import net.minecraft.core.BlockPos;
import net.minecraft.nbt.CompoundTag;
import net.minecraft.resources.ResourceKey;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.world.level.Level;
import net.minecraft.world.level.block.entity.BlockEntity;
import net.minecraft.world.level.block.state.BlockState;

public class RitualAltarBlockEntity extends BlockEntity {

    private ResourceKey<Level> targetDimension;

    public RitualAltarBlockEntity(BlockPos pos, BlockState state) {
        super(DimensionsRegistries.BLOCK_ENTITIES.getEntries().stream()
                .filter(r -> r.getId().getPath().equals("ritual_altar")).findFirst().get().get(), pos, state);
    }

    public void setTargetDimension(ResourceKey<Level> dim) {
        this.targetDimension = dim;
        setChanged();
    }
    
    public ResourceKey<Level> getTargetDimension() {
        return targetDimension;
    }

    @Override
    protected void saveAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.saveAdditional(tag, registries);
        if (targetDimension != null) {
            tag.putString("TargetDimension", targetDimension.location().toString());
        }
    }

    @Override
    protected void loadAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.loadAdditional(tag, registries);
        if (tag.contains("TargetDimension")) {
            this.targetDimension = ResourceKey.create(net.minecraft.core.registries.Registries.DIMENSION, ResourceLocation.parse(tag.getString("TargetDimension")));
        }
    }
}

package com.horizonpack.horizondimensions.events;

import com.horizonpack.horizondimensions.core.HorizonDimensions;
import com.horizonpack.horizondimensions.systems.MagicEngine;
import com.horizonpack.horizondimensions.systems.CorruptionEngine;
import net.minecraft.server.level.ServerPlayer;
import net.neoforged.bus.api.SubscribeEvent;
import net.neoforged.fml.common.Mod;
import net.neoforged.neoforge.event.tick.PlayerTickEvent;

@Mod.EventBusSubscriber(modid = HorizonDimensions.MODID)
public class DimensionsEventHandlers {

    @SubscribeEvent
    public static void onPlayerTick(PlayerTickEvent.Post event) {
        if (!event.getEntity().level().isClientSide && event.getEntity() instanceof ServerPlayer player) {
            if (player.tickCount % 40 == 0) { // Every 2 seconds
                MagicEngine.tickManaRegen(player);
            }
            if (player.tickCount % 100 == 0) { // Every 5 seconds
                CorruptionEngine.tickCorruptionEffects(player);
            }
        }
    }
}

package com.horizonpack.horizondimensions.network.packets;

import com.horizonpack.horizondimensions.core.HorizonDimensions;
import com.horizonpack.horizondimensions.systems.MagicEngine;
import io.netty.buffer.ByteBuf;
import net.minecraft.network.codec.ByteBufCodecs;
import net.minecraft.network.codec.StreamCodec;
import net.minecraft.network.protocol.common.custom.CustomPacketPayload;
import net.minecraft.resources.ResourceLocation;
import net.neoforged.neoforge.network.handling.IPayloadContext;

public record CastSpellPacket(ResourceLocation spellId) implements CustomPacketPayload {

    public static final Type<CastSpellPacket> TYPE = new Type<>(ResourceLocation.fromNamespaceAndPath(HorizonDimensions.MODID, "cast_spell"));

    public static final StreamCodec<ByteBuf, CastSpellPacket> STREAM_CODEC = StreamCodec.composite(
        ResourceLocation.STREAM_CODEC, CastSpellPacket::spellId,
        CastSpellPacket::new
    );

    @Override
    public Type<CastSpellPacket> type() { return TYPE; }

    public static void handleServer(CastSpellPacket packet, IPayloadContext context) {
        context.enqueueWork(() -> {
            net.minecraft.server.level.ServerPlayer player = (net.minecraft.server.level.ServerPlayer) context.player();
            MagicEngine.castSpell(player, packet.spellId());
        });
    }
}

package com.horizonpack.horizondimensions.core;

import net.neoforged.neoforge.common.ModConfigSpec;

public class DimensionsConfig {
    public static final ModConfigSpec COMMON_SPEC;
    public static final ModConfigSpec CLIENT_SPEC;

    public static class Common {
        public final ModConfigSpec.DoubleValue corruptionMultiplier;
        public final ModConfigSpec.BooleanValue permadeathAtMaxCorruption;
        public final ModConfigSpec.BooleanValue allowDimensionalRifts;

        Common(ModConfigSpec.Builder builder) {
            builder.push("magic");
            corruptionMultiplier = builder.comment("Scales corruption gained from spells").defineInRange("corruptionMultiplier", 1.0, 0.0, 10.0);
            permadeathAtMaxCorruption = builder.comment("If true, reaching BEYOND_SAVING deletes the player file").define("permadeathAtMaxCorruption", false);
            builder.pop();

            builder.push("dimensions");
            allowDimensionalRifts = builder.comment("Allow players to tear reality and create portals").define("allowDimensionalRifts", true);
            builder.pop();
        }
    }

    public static class Client {
        public final ModConfigSpec.BooleanValue showManaHUD;

        Client(ModConfigSpec.Builder builder) {
            builder.push("gui");
            showManaHUD = builder.comment("Show mana and corruption bars on HUD").define("showManaHUD", true);
            builder.pop();
        }
    }

    public static final Common COMMON;
    public static final Client CLIENT;

    static {
        var commonPair = new ModConfigSpec.Builder().configure(Common::new);
        COMMON = commonPair.getLeft();
        COMMON_SPEC = commonPair.getRight();

        var clientPair = new ModConfigSpec.Builder().configure(Client::new);
        CLIENT = clientPair.getLeft();
        CLIENT_SPEC = clientPair.getRight();
    }
}

27. Critical Implementation Notes
Dimension Serialization: Dimensions in 1.21.1 are solely defined by data packs, not Java code. The DimensionKeys.java class exists purely to provide ResourceKey<Level> references for the TeleportationHelper and RitualAltarBlockEntity to utilize when interacting with ServerLevel. Do not attempt to register DimensionType via DeferredRegister.
Corruption Tick Optimization: The CorruptionEngine checks corruption thresholds every 100 ticks (5 seconds). Spawning phantoms or teleporting players randomly at stage 4 (REALITY_BROKEN) must implement a cooldown check inside the player attachment data to prevent overwhelming the server or soft-locking the player in infinite teleport loops.
Spell Projectile Rendering: The FireBoltEntity and IceShardEntity extend AbstractHurtingProjectile. To render them correctly without missing texture boxes, the DimensionsEntityRenderer must register custom models utilizing EntityRenderersEvent.RegisterRenderers via the modEventBus.
Ley Line Seed Determinism: LeyLineEngine.calculateBaseIntensity uses a hash of the chunk coordinates combined with the world seed. This guarantees that ley lines are geographically fixed and players cannot "reroll" an area by unloading chunks.
Reagent Consumption Safety: MagicEngine.consumeReagent searches the player's inventory for an item matching a dynamic TagKey. This function is manually protected by a player.isCreative() check to ensure creative mode players do not have their hotbar items deleted while testing spells.
Boss Entities & Culling: Mythical entities like DimensionEaterEntity have a massive hitbox (10.0f x 10.0f). The clientTrackingRange in MythicalEntitiesRegistry must be explicitly set to 64 or 128 to prevent the boss from disappearing when the player backs away while the boss's center point is off-camera.
Attachment Persistence: PlayerMagicData holds the unlockedSpells Set and corruptionLevel. By using .copyOnDeath() on the attachment builder, corruption is deliberately not cleansed by dying, forcing the player to engage with the actual purification mechanics (e.g., Divine Magic rituals) rather than using death as an exploit.
Permadeath Override: If permadeathAtMaxCorruption is enabled in DimensionsConfig, reaching 100.0f corruption intercepts the LivingDeathEvent. Instead of normal respawn logic, the player's data file is wiped or their gamemode is permanently forced to Spectator, acting as a localized hardcore failure state.
