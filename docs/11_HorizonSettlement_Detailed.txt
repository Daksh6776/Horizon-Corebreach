1. Overview & Responsibilities
HorizonSettlement dictates how humanity builds and organizes its living spaces across the ages. It transforms random block placement into structured architectural feats, managed cities, and networked infrastructure.
Architecture: 14 distinct building styles tracking cultural evolution, defining load-bearing requirements, necessary construction materials, and the aesthetic layout of NPC structures.
Urban Planning: 12 specific city district types managing localized population density, property values, crime multipliers, and utility consumption.
Infrastructure: 10 core infrastructure types connecting districts and settlements. From primitive aqueducts to subway tunnels, these networks propagate water, electricity, and sanitation.
Mega-Structures: 9 monumental construction projects requiring massive labor, scaffolding, and years to complete, granting civilization-wide bonuses.
Construction Mechanics: Blueprints, scaffolding, and material delivery networks. Buildings are no longer instantly placed; they require dedicated ConstructionSiteBlockEntity processing over time.

import net.neoforged.fml.common.Mod;
import net.neoforged.fml.config.ModConfig;
import net.neoforged.fml.event.lifecycle.FMLCommonSetupEvent;
import net.neoforged.neoforge.common.NeoForge;

@Mod(HorizonSettlement.MODID)
public class HorizonSettlement {
    public static final String MODID = "horizonsettlement";
    public static final String VERSION = "1.0.0";

    public HorizonSettlement(IEventBus modEventBus, ModContainer modContainer) {
        modContainer.registerConfig(ModConfig.Type.COMMON, SettlementConfig.COMMON_SPEC);
        modContainer.registerConfig(ModConfig.Type.CLIENT, SettlementConfig.CLIENT_SPEC);

        SettlementRegistries.BLOCKS.register(modEventBus);
        SettlementRegistries.ITEMS.register(modEventBus);
        SettlementRegistries.BLOCK_ENTITIES.register(modEventBus);
        SettlementRegistries.MENU_TYPES.register(modEventBus);

        SettlementRegistries.BUILDING_STYLES.register(modEventBus);
        SettlementRegistries.CITY_DISTRICTS.register(modEventBus);
        SettlementRegistries.INFRASTRUCTURE_TYPES.register(modEventBus);
        SettlementRegistries.MEGA_STRUCTURES.register(modEventBus);

        modEventBus.addListener(this::commonSetup);
        modEventBus.addListener(SettlementRegistries::registerNewRegistries);
        modEventBus.addListener(SettlementPacketHandler::register);

        NeoForge.EVENT_BUS.register(SettlementEventHandlers.class);
    }

    private void commonSetup(FMLCommonSetupEvent event) {
        event.enqueueWork(() -> {
            UrbanZoningEngine.init();
            ConstructionEngine.init();
            UtilityNetworkEngine.init();
        });
    }
}

package com.horizonpack.horizonsettlement.systems;

import com.horizonpack.horizonsettlement.data.SettlementSavedData;
import net.minecraft.core.BlockPos;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.world.entity.player.Player;
import java.util.UUID;

public class UrbanZoningEngine {

    public static void init() {}

    /**
     * Designates a 16x16 chunk area as a specific City District.
     */
    public static void zoneDistrict(ServerLevel level, BlockPos pos, ResourceLocation districtId, UUID civilizationId, Player player) {
        SettlementSavedData data = SettlementSavedData.get(level);
        
        long chunkKey = (long)(pos.getX() >> 4) << 32 | (pos.getZ() >> 4 & 0xFFFFFFFFL);
        data.setDistrict(chunkKey, districtId, civilizationId);
        
        // Notify Horizonsociety that a new district was zoned, which updates pop caps
        net.neoforged.neoforge.common.NeoForge.EVENT_BUS.post(new com.horizonpack.horizonsettlement.events.custom.DistrictFormedEvent(level, chunkKey, districtId, civilizationId));
        
        player.displayClientMessage(net.minecraft.network.chat.Component.translatable("message.horizonsettlement.district_zoned", districtId.getPath()), true);
    }

    /**
     * Evaluates land value and gentrification forces. Called daily by HorizonCore's DayAdvanceEvent.
     */
    public static void tickZoning(ServerLevel level) {
        SettlementSavedData data = SettlementSavedData.get(level);
        
        data.getAllDistricts().forEach((chunkKey, districtId) -> {
            float baseValue = getDistrictValue(level, districtId);
            float infrastructureScore = UtilityNetworkEngine.evaluateInfrastructure(level, chunkKey);
            
            float newValue = baseValue * (1.0f + infrastructureScore);
            data.setLandValue(chunkKey, newValue);
            
            // If land value exceeds slum threshold significantly, trigger gentrification
            if (districtId.getPath().equals("slum") && newValue > 2.0f) {
                data.setDistrict(chunkKey, ResourceLocation.fromNamespaceAndPath("horizonsettlement", "residential"), null);
            }
        });
    }

    private static float getDistrictValue(ServerLevel level, ResourceLocation districtId) {
        var district = level.registryAccess().registryOrThrow(com.horizonpack.horizonsettlement.core.SettlementRegistries.DISTRICT_KEY).get(districtId);
        return district != null ? district.propertyValueMultiplier() : 1.0f;
    }
}

import com.horizonpack.horizonsettlement.blockentities.ConstructionSiteBlockEntity;
import net.minecraft.core.BlockPos;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.world.item.ItemStack;

public class ConstructionEngine {

    public static void init() {}

    /**
     * Processes ticks on construction sites. Converts stored materials into placed blocks.
     */
    public static void tickConstructionSite(ServerLevel level, BlockPos pos, ConstructionSiteBlockEntity be) {
        if (be.isComplete()) return;

        // Requires active labor from players or assigned NPC workers
        if (be.getActiveWorkers() > 0) {
            be.advanceProgress(be.getActiveWorkers());
            
            if (be.readyForNextBlock()) {
                BlockPos targetPos = be.getNextBlueprintPosition();
                ItemStack requiredMaterial = be.getRequiredMaterialFor(targetPos);
                
                if (be.consumeMaterial(requiredMaterial)) {
                    // Place scaffolding temporarily if above ground
                    if (targetPos.getY() > pos.getY() + 2) {
                        level.setBlock(targetPos.below(), com.horizonpack.horizonsettlement.core.SettlementRegistries.SCAFFOLDING_BLOCK.get().defaultBlockState(), 3);
                    }
                    
                    // Place the actual block
                    level.setBlock(targetPos, be.getBlockStateFor(targetPos), 3);
                    be.markBlockPlaced(targetPos);
                } else {
                    // Halt construction, missing materials
                    be.setStatus("Waiting for materials: " + requiredMaterial.getHoverName().getString());
                }
            }
            
            if (be.checkCompletion()) {
                // Cleanup scaffolding
                cleanupScaffolding(level, pos, be.getBounds());
                level.removeBlock(pos, false); // Remove the construction site block
            }
        }
    }
    
    private static void cleanupScaffolding(ServerLevel level, BlockPos origin, net.minecraft.world.phys.AABB bounds) {
        BlockPos.betweenClosedStream(bounds).forEach(p -> {
            if (level.getBlockState(p).is(com.horizonpack.horizonsettlement.core.SettlementRegistries.SCAFFOLDING_BLOCK.get())) {
                level.removeBlock(p, false);
            }
        });
    }
}

package com.horizonpack.horizonsettlement.systems;

import net.minecraft.core.BlockPos;
import net.minecraft.server.level.ServerLevel;
import java.util.HashSet;
import java.util.Set;
import java.util.ArrayDeque;
import java.util.Queue;

public class UtilityNetworkEngine {

    public static void init() {}

    /**
     * Returns a 0.0 to 1.0 score based on water, sewer, and power access.
     */
    public static float evaluateInfrastructure(ServerLevel level, long chunkKey) {
        int x = (int) (chunkKey >> 32) * 16 + 8;
        int z = (int) chunkKey * 16 + 8;
        BlockPos center = new BlockPos(x, 64, z);
        
        boolean hasWater = hasUtilityAccess(level, center, "water", 32);
        boolean hasSewer = hasUtilityAccess(level, center, "sewer", 32);
        boolean hasPower = hasUtilityAccess(level, center, "power", 64);
        
        float score = 0.0f;
        if (hasWater) score += 0.4f;
        if (hasSewer) score += 0.3f;
        if (hasPower) score += 0.3f;
        
        return score;
    }

    private static boolean hasUtilityAccess(ServerLevel level, BlockPos pos, String utilityType, int radius) {
        // Simplified BFS search for UtilityNodeBlockEntity matching the type
        Set<BlockPos> visited = new HashSet<>();
        Queue<BlockPos> queue = new ArrayDeque<>();
        queue.add(pos);
        
        int steps = 0;
        while(!queue.isEmpty() && steps < radius) {
            BlockPos current = queue.poll();
            if (visited.contains(current)) continue;
            visited.add(current);
            steps++;
            
            net.minecraft.world.level.block.entity.BlockEntity be = level.getBlockEntity(current);
            if (be instanceof com.horizonpack.horizonsettlement.blockentities.UtilityNodeBlockEntity node) {
                if (node.provides(utilityType)) return true;
            }
            
            for (net.minecraft.core.Direction dir : net.minecraft.core.Direction.values()) {
                queue.add(current.relative(dir));
            }
        }
        return false;
    }
}

package com.horizonpack.horizonsettlement.data;

import net.minecraft.nbt.CompoundTag;
import net.minecraft.nbt.ListTag;
import net.minecraft.nbt.Tag;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.world.level.saveddata.SavedData;
import org.jetbrains.annotations.NotNull;

import java.util.HashMap;
import java.util.Map;
import java.util.UUID;

public class SettlementSavedData extends SavedData {

    public static final String DATA_NAME = "horizonsettlement_state";

    // ChunkKey -> District ID
    private final Map<Long, ResourceLocation> zonedDistricts = new HashMap<>();
    // ChunkKey -> Civ UUID
    private final Map<Long, UUID> districtOwners = new HashMap<>();
    // ChunkKey -> Land Value float
    private final Map<Long, Float> landValues = new HashMap<>();

    public static SettlementSavedData get(ServerLevel level) {
        return level.getDataStorage().computeIfAbsent(
                new SavedData.Factory<>(SettlementSavedData::new, SettlementSavedData::load),
                DATA_NAME
        );
    }

    public void setDistrict(long chunkKey, ResourceLocation districtId, UUID ownerId) {
        zonedDistricts.put(chunkKey, districtId);
        if (ownerId != null) districtOwners.put(chunkKey, ownerId);
        setDirty();
    }

    public Map<Long, ResourceLocation> getAllDistricts() {
        return zonedDistricts;
    }

    public void setLandValue(long chunkKey, float value) {
        landValues.put(chunkKey, value);
        setDirty();
    }

    @Override
    public @NotNull CompoundTag save(@NotNull CompoundTag tag) {
        ListTag zones = new ListTag();
        zonedDistricts.forEach((chunkKey, districtId) -> {
            CompoundTag zTag = new CompoundTag();
            zTag.putLong("Chunk", chunkKey);
            zTag.putString("District", districtId.toString());
            if (districtOwners.containsKey(chunkKey)) zTag.putUUID("Owner", districtOwners.get(chunkKey));
            if (landValues.containsKey(chunkKey)) zTag.putFloat("Value", landValues.get(chunkKey));
            zones.add(zTag);
        });
        tag.put("Zoning", zones);
        return tag;
    }

    public static SettlementSavedData load(CompoundTag tag) {
        SettlementSavedData data = new SettlementSavedData();
        ListTag zones = tag.getList("Zoning", Tag.TAG_COMPOUND);
        for (int i = 0; i < zones.size(); i++) {
            CompoundTag zTag = zones.getCompound(i);
            long chunk = zTag.getLong("Chunk");
            data.zonedDistricts.put(chunk, ResourceLocation.parse(zTag.getString("District")));
            if (zTag.hasUUID("Owner")) data.districtOwners.put(chunk, zTag.getUUID("Owner"));
            if (zTag.contains("Value")) data.landValues.put(chunk, zTag.getFloat("Value"));
        }
        return data;
    }
}

package com.horizonpack.horizonsettlement.blockentities;

import com.horizonpack.horizonsettlement.core.SettlementRegistries;
import net.minecraft.core.BlockPos;
import net.minecraft.nbt.CompoundTag;
import net.minecraft.world.item.ItemStack;
import net.minecraft.world.level.block.entity.BlockEntity;
import net.minecraft.world.level.block.state.BlockState;
import net.minecraft.world.phys.AABB;

import java.util.HashMap;
import java.util.Map;

public class ConstructionSiteBlockEntity extends BlockEntity {

    private int activeWorkers = 0;
    private int totalProgress = 0;
    private int requiredProgress = 1000;
    private String statusMsg = "Awaiting Blueprint";
    private ResourceLocation blueprintId;

    // Item ID string to count remaining
    private final Map<String, Integer> requiredMaterials = new HashMap<>();

    public ConstructionSiteBlockEntity(BlockPos pos, BlockState state) {
        super(SettlementRegistries.CONSTRUCTION_SITE_BE.get(), pos, state);
    }

    public int getActiveWorkers() { return activeWorkers; }
    public void setActiveWorkers(int count) { this.activeWorkers = count; }

    public void advanceProgress(int amount) {
        this.totalProgress += amount;
        setChanged();
    }

    public boolean isComplete() { return totalProgress >= requiredProgress; }
    
    public boolean readyForNextBlock() {
        return totalProgress % 10 == 0; // Places a block every 10 progress ticks
    }

    public BlockPos getNextBlueprintPosition() {
        // Reads from loaded blueprint schematics. Returns a dummy pos here.
        return this.worldPosition.above();
    }

    public ItemStack getRequiredMaterialFor(BlockPos pos) {
        return new ItemStack(net.minecraft.world.level.block.Blocks.OAK_PLANKS);
    }

    public boolean consumeMaterial(ItemStack stack) {
        String key = net.minecraft.core.registries.BuiltInRegistries.ITEM.getKey(stack.getItem()).toString();
        if (requiredMaterials.getOrDefault(key, 0) > 0) {
            requiredMaterials.put(key, requiredMaterials.get(key) - 1);
            setChanged();
            return true;
        }
        return false; // Not in local storage
    }

    public BlockState getBlockStateFor(BlockPos pos) {
        return net.minecraft.world.level.block.Blocks.OAK_PLANKS.defaultBlockState();
    }

    public void markBlockPlaced(BlockPos pos) {}
    
    public void setStatus(String msg) {
        this.statusMsg = msg;
        setChanged();
    }

    public AABB getBounds() {
        return new AABB(this.worldPosition).inflate(10); // Example 20x20x20 area
    }

    public boolean checkCompletion() {
        return isComplete();
    }

    @Override
    protected void saveAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.saveAdditional(tag, registries);
        tag.putInt("Workers", activeWorkers);
        tag.putInt("Progress", totalProgress);
        tag.putInt("ReqProgress", requiredProgress);
        tag.putString("Status", statusMsg);
        
        CompoundTag mats = new CompoundTag();
        requiredMaterials.forEach(mats::putInt);
        tag.put("Materials", mats);
    }

    @Override
    protected void loadAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.loadAdditional(tag, registries);
        activeWorkers = tag.getInt("Workers");
        totalProgress = tag.getInt("Progress");
        requiredProgress = tag.getInt("ReqProgress");
        statusMsg = tag.getString("Status");
        
        CompoundTag mats = tag.getCompound("Materials");
        requiredMaterials.clear();
        for (String key : mats.getAllKeys()) {
            requiredMaterials.put(key, mats.getInt(key));
        }
    }
}

package com.horizonpack.horizonsettlement.blockentities;

import com.horizonpack.horizonsettlement.core.SettlementRegistries;
import net.minecraft.core.BlockPos;
import net.minecraft.nbt.CompoundTag;
import net.minecraft.world.level.block.entity.BlockEntity;
import net.minecraft.world.level.block.state.BlockState;

public class UtilityNodeBlockEntity extends BlockEntity {

    private boolean providesWater = false;
    private boolean providesSewer = false;
    private boolean providesPower = false;

    public UtilityNodeBlockEntity(BlockPos pos, BlockState state) {
        super(SettlementRegistries.UTILITY_NODE_BE.get(), pos, state);
        
        // Auto-configure based on the block type attached to this entity
        if (state.is(SettlementRegistries.AQUEDUCT_CHANNEL.get())) providesWater = true;
        if (state.is(SettlementRegistries.SEWER_PIPE.get())) providesSewer = true;
        if (state.is(SettlementRegistries.ELECTRICAL_GRID_POLE.get())) providesPower = true;
    }

    public boolean provides(String type) {
        return switch(type) {
            case "water" -> providesWater;
            case "sewer" -> providesSewer;
            case "power" -> providesPower;
            default -> false;
        };
    }

    @Override
    protected void saveAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.saveAdditional(tag, registries);
        tag.putBoolean("Water", providesWater);
        tag.putBoolean("Sewer", providesSewer);
        tag.putBoolean("Power", providesPower);
    }

    @Override
    protected void loadAdditional(CompoundTag tag, net.minecraft.core.HolderLookup.Provider registries) {
        super.loadAdditional(tag, registries);
        providesWater = tag.getBoolean("Water");
        providesSewer = tag.getBoolean("Sewer");
        providesPower = tag.getBoolean("Power");
    }
}

package com.horizonpack.horizonsettlement.core;

import net.neoforged.neoforge.common.ModConfigSpec;

public class SettlementConfig {
    public static final ModConfigSpec COMMON_SPEC;
    public static final ModConfigSpec CLIENT_SPEC;

    public static class Common {
        public final ModConfigSpec.DoubleValue constructionSpeedMultiplier;
        public final ModConfigSpec.BooleanValue requireInfrastructure;
        public final ModConfigSpec.IntValue maxMegaStructuresPerCiv;

        Common(ModConfigSpec.Builder builder) {
            builder.push("construction");
            constructionSpeedMultiplier = builder.comment("Scales speed of NPC and Blueprint construction.").defineInRange("constructionSpeedMultiplier", 1.0, 0.1, 10.0);
            requireInfrastructure = builder.comment("Districts without water/sewer/power suffer massive penalties.").define("requireInfrastructure", true);
            maxMegaStructuresPerCiv = builder.comment("Limit on wonders built per civilization.").defineInRange("maxMegaStructuresPerCiv", 3, 1, 10);
            builder.pop();
        }
    }

    public static class Client {
        public final ModConfigSpec.BooleanValue showZoningOverlay;

        Client(ModConfigSpec.Builder builder) {
            builder.push("gui");
            showZoningOverlay = builder.comment("Render colored chunk borders when holding Zoning Tool.").define("showZoningOverlay", true);
            builder.pop();
        }
    }

    public static final Common COMMON;
    public static final Client CLIENT;

    static {
        var commonPair = new ModConfigSpec.Builder().configure(Common::new);
        COMMON = commonPair.getLeft();
        COMMON_SPEC = commonPair.getRight();

        var clientPair = new ModConfigSpec.Builder().configure(Client::new);
        CLIENT = clientPair.getLeft();
        CLIENT_SPEC = clientPair.getRight();
    }

25. Critical Implementation Notes
Chunk-Based Zoning Serialization: UrbanZoningEngine maps districts via a long chunk key inside SettlementSavedData. To prevent severe memory bloat on large servers, only map chunks that are explicitly designated. Natural, un-zoned chunks simply return null and are treated as wilderness.
Construction Site Processing Rate: A ConstructionSiteBlockEntity places physical blocks into the world. Firing level.setBlock continuously causes intense chunk meshing lag. The engine places blocks at a controlled modulus rate (totalProgress % 10 == 0), artificially spreading out rendering workload over several seconds.
Scaffolding Cleanup: cleanupScaffolding uses a 3D volume scan BlockPos.betweenClosedStream(bounds) which can lock the main thread if the boundary is enormous (e.g., a skyscraper). Limit bounds lookup to 32x32x32 blocks per tick, saving state via NBT to resume cleanup on subsequent ticks.
Utility Pathfinding Extent: The BFS search inside UtilityNetworkEngine.hasUtilityAccess checks a radius of 32-64 blocks. To prevent extreme lag spikes during zoning calculations, this BFS is restricted by a maximum step counter and only traverses block entities registered under the infrastructure tag, skipping solid non-utility blocks instantly.
Gentrification Override Logic: The tickZoning function can autonomously rewrite a chunk's registered district ID (e.g., Slum -> Residential) based on infrastructure score. This requires pushing SyncZoningDataPacket to tracking clients immediately, or their client-side ZoningOverlayRenderer will fall out of sync with the server's land value representation.
Mega-Structure Requirements: Mega-structures like the space_elevator define requirements of 50,000 workers. ConstructionEngine multiplies activeWorkers linearly to increment progress. If ConstructionSiteBlockEntity lacks a worker assignment pipeline (via horizonsociety NPC integration), a single player will take real-world months to finish the structure.
Infrastructure Cross-Mod Connections: Blocks like electrical_grid_pole provide zoning utility points and simultaneously expose Capabilities.EnergyStorage.BLOCK from horizonpower. They are hybrid nodes. Do not create isolated networks; use the existing PowerGridNode logic to route RF, simply querying presence for the hasPower check.
Blueprint Item NBT Data Components: The BlueprintItem holds schematic data defining what the ConstructionSiteBlockEntity must build. Under 1.21.1 rules, this data must be wrapped in a registered DataComponentType (e.g., horizonsettlement:schematic_data), avoiding the deprecated getOrCreateTag() method.
