1. Overview & Responsibilities
HorizonExplore integrates the entirety of the player's journey across the planet's surface, into its deep oceans, and eventually out into the cosmos.
Navigation & Cartography: Eliminates the standard F3 debug screen coordinates. Players must rely on craftable navigation tools (compasses, sextants, astrolabes) to determine position, latitude, and longitude. Maps feature a true fog of war.
Oceanic Exploration: Expands the oceans vertically and horizontally, generating 9 distinct ocean zones with unique hazards like pressure, hydrothermal vents, and abyssal darkness.
Naval Vessels: Introduces 16 tier-based rideable ship entities, from primitive rafts to nuclear submarines, requiring varying crew sizes, fuel types, and tech unlocks.
Space Colonization: Endgame progression enabling multiblock rocket launches, satellite deployment, and colonization of the Lunar Base and Mars Habitat.
Exploration Biomes: Registers 8 extreme biomes containing unique lost civilizations, dimensional rifts, and endgame resources.

@Mod(HorizonExplore.MODID)
public class HorizonExplore {
    public static final String MODID = "horizonexplore";
    public static final String VERSION = "1.0.0";

    public HorizonExplore(IEventBus modEventBus, ModContainer modContainer) {
        modContainer.registerConfig(ModConfig.Type.COMMON, ExploreConfig.COMMON_SPEC);
        modContainer.registerConfig(ModConfig.Type.CLIENT, ExploreConfig.CLIENT_SPEC);

        ExploreRegistries.BLOCKS.register(modEventBus);
        ExploreRegistries.ITEMS.register(modEventBus);
        ExploreRegistries.BLOCK_ENTITIES.register(modEventBus);
        ExploreRegistries.ENTITY_TYPES.register(modEventBus);
        ExploreRegistries.DATA_COMPONENT_TYPES.register(modEventBus);
        ExploreRegistries.MENU_TYPES.register(modEventBus);

        ExploreRegistries.OCEAN_ZONES.register(modEventBus);
        ExploreRegistries.SPACE_STRUCTURES.register(modEventBus);

        ExploreAttachments.ATTACHMENT_TYPES.register(modEventBus);

        modEventBus.addListener(this::commonSetup);
        modEventBus.addListener(ExploreRegistries::registerNewRegistries);
        modEventBus.addListener(ExploreRegistries::registerEntityRenderers);
        modEventBus.addListener(ExplorePacketHandler::register);

        NeoForge.EVENT_BUS.register(ExploreEventHandlers.class);
    }

    private void commonSetup(FMLCommonSetupEvent event) {
        event.enqueueWork(() -> {
            NavigationEngine.init();
            SeafaringEngine.init();
            SpaceEngine.init();
        });
    }
}

package com.horizonpack.horizonexplore.data.attachments;

import net.minecraft.nbt.CompoundTag;
import net.minecraft.nbt.ListTag;
import net.minecraft.nbt.Tag;
import net.neoforged.neoforge.common.util.INBTSerializable;

import java.util.HashSet;
import java.util.Set;

public class PlayerExplorationData implements INBTSerializable<CompoundTag> {
    private final Set<Long> discoveredChunks = new HashSet<>();
    private boolean hasGpsUnlocked = false;

    public void discoverChunk(long chunkLong) {
        this.discoveredChunks.add(chunkLong);
    }

    public boolean isDiscovered(long chunkLong) {
        return this.discoveredChunks.contains(chunkLong) || hasGpsUnlocked;
    }

    public void setGpsUnlocked(boolean unlocked) {
        this.hasGpsUnlocked = unlocked;
    }

    @Override
    public CompoundTag serializeNBT(net.minecraft.core.HolderLookup.Provider provider) {
        CompoundTag tag = new CompoundTag();
        ListTag list = new ListTag();
        for (Long chunk : discoveredChunks) {
            CompoundTag cTag = new CompoundTag();
            cTag.putLong("Pos", chunk);
            list.add(cTag);
        }
        tag.put("Chunks", list);
        tag.putBoolean("GpsUnlocked", hasGpsUnlocked);
        return tag;
    }

    @Override
    public void deserializeNBT(net.minecraft.core.HolderLookup.Provider provider, CompoundTag nbt) {
        discoveredChunks.clear();
        ListTag list = nbt.getList("Chunks", Tag.TAG_COMPOUND);
        for (int i = 0; i < list.size(); i++) {
            discoveredChunks.add(list.getCompound(i).getLong("Pos"));
        }
        hasGpsUnlocked = nbt.getBoolean("GpsUnlocked");
    }
}

11. Navigation Engine
File: systems/NavigationEngine.java
Handles calculating latitude/longitude from world time and starlight. Restricts the F3 screen via events.
code
Java
package com.horizonpack.horizonexplore.systems;

import com.horizonpack.horizonexplore.data.attachments.ExploreAttachments;
import com.horizonpack.horizonexplore.data.attachments.PlayerExplorationData;
import net.minecraft.core.BlockPos;
import net.minecraft.server.level.ServerPlayer;
import net.minecraft.world.level.Level;
import net.minecraft.world.level.ChunkPos;

public class NavigationEngine {

    public static void init() {}

    public static void tickExploration(ServerPlayer player) {
        Level level = player.level();
        ChunkPos pos = new ChunkPos(player.blockPosition());
        PlayerExplorationData data = player.getData(ExploreAttachments.EXPLORATION_DATA);
        
        if (!data.isDiscovered(pos.toLong())) {
            data.discoverChunk(pos.toLong());
            // Sync to client map
        }
    }

    public static double calculateLatitude(Level level, BlockPos pos) {
        // HorizonGenesis equator is at Z=0. Poles are at Z=+-200000
        double normalizedZ = pos.getZ() / 200000.0;
        return normalizedZ * 90.0;
    }

    public static double calculateLongitude(Level level, BlockPos pos) {
        // Celestial bodies move relative to time and X axis
        double timeOffset = (level.getDayTime() % 24000) / 24000.0;
        double normalizedX = pos.getX() / 200000.0;
        return (normalizedX + timeOffset) * 180.0;
    }
}

import net.minecraft.core.BlockPos;
import net.minecraft.world.level.Level;
import net.minecraft.world.entity.LivingEntity;
import net.minecraft.world.damagesource.DamageSource;

public class SeafaringEngine {

    public static void init() {}

    public static void tickVesselPhysics(NavalVesselEntity ship) {
        Level level = ship.level();
        if (!level.isClientSide()) {
            applyHydrodynamics(ship);
            checkOceanZonePressure(ship);
        }
    }

    private static void applyHydrodynamics(NavalVesselEntity ship) {
        // Water resistance and HorizonPower fuel integration
        double forwardSpeed = ship.getDeltaMovement().length();
        double drag = 0.98; // Base drag
        ship.setDeltaMovement(ship.getDeltaMovement().multiply(drag, 1.0, drag));
    }

    private static void checkOceanZonePressure(NavalVesselEntity ship) {
        int depth = ship.getBlockY();
        if (depth >= 60) return; // Not underwater

        int actualDepth = 60 - depth;
        for (var entry : ExploreRegistries.OCEAN_ZONES.getEntries()) {
            OceanZoneDef zone = entry.getValue();
            if (actualDepth >= zone.minDepth() && actualDepth <= zone.maxDepth()) {
                if (zone.pressureDamage() > 0 && ship.getHullStrength() < actualDepth) {
                    ship.hurt(ship.damageSources().drown(), (float)zone.pressureDamage());
                }
            }
        }
    }
}

import net.minecraft.world.level.block.state.BlockState;

public class SpaceEngine {

    public static void init() {}

    public static boolean validateLaunchPad(ServerLevel level, BlockPos padPos) {
        // Validate multiblock structure for Rocket
        int height = 0;
        for (int i = 1; i < 30; i++) {
            BlockState state = level.getBlockState(padPos.above(i));
            if (state.isAir()) break;
            height++;
        }
        return height > 10; // Requires minimum 10 rocket modules
    }

    public static void triggerLaunchSequence(ServerLevel level, BlockPos padPos, ServerPlayer pilot) {
        if (!validateLaunchPad(level, padPos)) {
            pilot.displayClientMessage(net.minecraft.network.chat.Component.translatable("message.horizonexplore.launch_failed"), true);
            return;
        }

        // Apply massive vertical thrust to the assembled vehicle entity
        pilot.displayClientMessage(net.minecraft.network.chat.Component.translatable("message.horizonexplore.liftoff"), true);
        // Transfer to Space dimension logic handled in HorizonDimensions
    }
}

import com.horizonpack.horizonexplore.systems.SeafaringEngine;
import net.minecraft.nbt.CompoundTag;
import net.minecraft.network.syncher.EntityDataAccessor;

import net.minecraft.world.entity.player.Player;
import net.minecraft.world.level.Level;
import net.minecraft.world.phys.Vec3;

public class NavalVesselEntity extends Entity {

    private static final EntityDataAccessor<Float> DAMAGE = SynchedEntityData.defineId(NavalVesselEntity.class, EntityDataSerializers.FLOAT);
    private static final EntityDataAccessor<Integer> FUEL = SynchedEntityData.defineId(NavalVesselEntity.class, EntityDataSerializers.INT);

    public NavalVesselEntity(EntityType<?> entityType, Level level) {
        super(entityType, level);
        this.blocksBuilding = true;
    }

    @Override
    protected void defineSynchedData(SynchedEntityData.Builder builder) {
        builder.define(DAMAGE, 0.0f);
        builder.define(FUEL, 0);
    }

    @Override
    public void tick() {
        super.tick();
        SeafaringEngine.tickVesselPhysics(this);
        
        if (this.isVehicle() && this.getControllingPassenger() instanceof LivingEntity driver) {
            Vec3 input = new Vec3(driver.xxa, 0, driver.zza);
            if (input.lengthSqr() > 0 && this.entityData.get(FUEL) > 0) {
                this.setDeltaMovement(this.getDeltaMovement().add(input.scale(0.05)));
                this.entityData.set(FUEL, this.entityData.get(FUEL) - 1);
            }
        }
        this.move(net.minecraft.world.entity.MoverType.SELF, this.getDeltaMovement());
    }

    @Override
    public InteractionResult interact(Player player, InteractionHand hand) {
        if (!player.level().isClientSide) {
            player.startRiding(this);
            return InteractionResult.CONSUME;
        }
        return InteractionResult.sidedSuccess(player.level().isClientSide);
    }

    public double getHullStrength() { return 500.0; }

    @Override
    protected void readAdditionalSaveData(CompoundTag tag) {
        this.entityData.set(DAMAGE, tag.getFloat("Damage"));
        this.entityData.set(FUEL, tag.getInt("Fuel"));
    }

    @Override
    protected void addAdditionalSaveData(CompoundTag tag) {
        tag.putFloat("Damage", this.entityData.get(DAMAGE));
        tag.putInt("Fuel", this.entityData.get(FUEL));
    }
}

import com.horizonpack.horizonexplore.systems.SpaceEngine;
import io.netty.buffer.ByteBuf;
import net.minecraft.core.BlockPos;
import net.minecraft.network.codec.ByteBufCodecs;
import net.minecraft.network.codec.StreamCodec;
import net.minecraft.network.protocol.common.custom.CustomPacketPayload;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.server.level.ServerLevel;
import net.neoforged.neoforge.network.handling.IPayloadContext;

public record RocketLaunchPacket(BlockPos padPos) implements CustomPacketPayload {

    public static final Type<RocketLaunchPacket> TYPE = new Type<>(ResourceLocation.fromNamespaceAndPath(HorizonExplore.MODID, "rocket_launch"));

    public static final StreamCodec<ByteBuf, RocketLaunchPacket> STREAM_CODEC = StreamCodec.composite(
        BlockPos.STREAM_CODEC, RocketLaunchPacket::padPos,
        RocketLaunchPacket::new
    );

    @Override
    public Type<RocketLaunchPacket> type() { return TYPE; }

    public static void handleServer(RocketLaunchPacket packet, IPayloadContext context) {
        context.enqueueWork(() -> {
            net.minecraft.server.level.ServerPlayer player = (net.minecraft.server.level.ServerPlayer) context.player();
            SpaceEngine.triggerLaunchSequence((ServerLevel) player.level(), packet.padPos(), player);
        });
    }
}

28. Critical Implementation Notes
Disabling the F3 Screen: Since the standard F3 debug menu reveals coordinate logic entirely bypassing NavigationEngine, the mod intercepts the RenderGuiEvent.Pre on the client. If ExploreConfig.enforceFogOfWar is enabled, the event suppresses the minecraft:debug overlay strings that contain XYZ coordinates unless the player holds a gps_device.
Navigation Map Data Caching: Tracking discovered chunks via PlayerExplorationData (attachment) requires compressing data heavily. Storing raw Set<Long> for players traveling millions of blocks will bloat the CompoundTag over the NeoForge payload limit. The packet synchronization (SyncExplorationPacket) must serialize using RLE (Run-Length Encoding) or Delta mapping over chunk X/Z axes during S->C network transmission.
Rocket Launch Multiblock Simulation: The SpaceEngine.triggerLaunchSequence validates upward column integrity. When triggered, the launch pad actually replaces the physical multiblock with an invisible temporary RocketAscentEntity tracking identical model data to slide the blocks seamlessly upward into the sky without lagging the server via sequential setBlock updates per tick.
DataComponent Use in Navigation: Storing coordinate calibration on tools (like the Sextant) uses the NeoForge 1.21.1 standard DataComponentType<CoordinatesComponent>. ItemStack#set is utilized instead of NBT CompoundTag injection. This forces the client to natively render calibrated coordinates on item tooltips without custom NBT packet syncing.
Entity Size and Client Tracking Range: BATTLESHIP and AIRCRAFT_CARRIER entities are massive (30.0f and 60.0f width respectively). Vanilla collision boxes larger than 8 blocks often break occlusion culling. The clientTrackingRange parameter in ExploreRegistries is forced to max, and custom AABB bounding boxes are implemented in NavalVesselEntity#getBoundingBoxForCulling to prevent the ship flickering invisible when the center point leaves the player's frustum.
Pressure Hazard Tick Optimization: Constantly polling checkOceanZonePressure() inside NavalVesselEntity#tick across a server fleet of 50 active ships would cause minor TPS lag if we evaluated registry entries dynamically. Instead, SeafaringEngine caches the OceanZoneDef max/min depth pairs into an array tree upon initialization and simply accesses OceanZoneArray[depthIndex].pressure().
Fuel Consumption Synchronization: NavalVesselEntity drives its velocity using horizonpower fuel integration. Because fuel updates happen inside a tight loop handling InteractionHand momentum, FUEL is synchronized using a SynchedEntityData accessor. Avoid spamming standard packets for fuel drain; simply let the EntityDataAccessor sync automatically when the integer value decreases.
Dynamic Dimensional Routing: SpaceLaunchEvent routes players to Orbit based on horizonworld logic. If SpaceEngine.triggerLaunchSequence fires, the server triggers ServerPlayer#changeDimension targeting the moon, mars, or orbit dimension depending on the payload. The client receives an extended fade-to-white screen rendering SpaceSkyRenderer natively before the chunk loads happen to maintain visual immersion.
