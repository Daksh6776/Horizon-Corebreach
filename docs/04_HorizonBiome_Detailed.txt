## 1. Overview & Responsibilities

HorizonBiome implements the **biological and ecological simulation** for the Horizon Corebreach modpack. It populates the geological canvas provided by `horizonworld` with interactive life.

- **Soil Systems:** 10 distinct soil types with specific nutrient retention, water capacity, and crop yield modifiers.
- **Flora & Crops:** 50 distinct plant species (wild plants, staple crops, and fruit trees) that obey climate zones, seasons, and soil mechanics.
- **Fauna & Wildlife:** 40 distinct animal species with complex AI, including predator-prey dynamics, migration, pack behavior, and a multi-stage domestication/breeding system.
- **Decomposition:** A dedicated Carcass entity system where dead animals progress through stages (fresh, decomposing, rotten, skeleton) attracting scavengers and generating disease vectors.
- **Microbiology:** 18 historically accurate diseases that infect players and NPCs via vectors (water, ticks, carcasses, weather, nutrition deficiency).

## 3. Mod Entry Point

**File:** `core/HorizonBiome.java`

```java
package com.horizonpack.horizonbiome.core;

import com.horizonpack.horizonbiome.events.BiomeEventHandlers;
import com.horizonpack.horizonbiome.network.BiomePacketHandler;
import com.horizonpack.horizonbiome.systems.DiseaseEngine;
import net.neoforged.bus.api.IEventBus;
import net.neoforged.fml.ModContainer;
import net.neoforged.fml.common.Mod;
import net.neoforged.fml.config.ModConfig;
import net.neoforged.fml.event.lifecycle.FMLCommonSetupEvent;
import net.neoforged.neoforge.common.NeoForge;

@Mod(HorizonBiome.MODID)
public class HorizonBiome {
    public static final String MODID = "horizonbiome";
    public static final String VERSION = "1.0.0";

    public HorizonBiome(IEventBus modEventBus, ModContainer modContainer) {
        modContainer.registerConfig(ModConfig.Type.COMMON, BiomeConfig.COMMON_SPEC);
        modContainer.registerConfig(ModConfig.Type.CLIENT, BiomeConfig.CLIENT_SPEC);

        BiomeRegistries.BLOCKS.register(modEventBus);
        BiomeRegistries.ITEMS.register(modEventBus);
        BiomeRegistries.ENTITY_TYPES.register(modEventBus);
        BiomeRegistries.CREATIVE_TABS.register(modEventBus);
        
        // Register custom diseases to HorizonCore's registry
        BiomeRegistries.DISEASES.register(modEventBus);

        modEventBus.addListener(this::commonSetup);
        modEventBus.addListener(BiomeRegistries::registerEntityAttributes);
        modEventBus.addListener(BiomePacketHandler::register);

        NeoForge.EVENT_BUS.register(BiomeEventHandlers.class);
    }

    private void commonSetup(FMLCommonSetupEvent event) {
        event.enqueueWork(() -> {
            DiseaseEngine.init();
            BiomeRegistries.registerCompostables();
        });
    }
}

package com.horizonpack.horizonbiome.data;

import net.minecraft.world.level.block.Block;

public record SoilNutrientData(
    float nitrogen,
    float phosphorus,
    float potassium,
    float waterRetention,
    float drainage,
    float baseCropYieldMultiplier
) {
    public static SoilNutrientData getFor(Block block) {
        String id = net.minecraft.core.registries.BuiltInRegistries.BLOCK.getKey(block).getPath();
        return switch(id) {
            case "clay_soil" -> new SoilNutrientData(0.4f, 0.5f, 0.6f, 0.9f, 0.1f, 0.8f);
            case "sandy_soil" -> new SoilNutrientData(0.2f, 0.2f, 0.3f, 0.2f, 0.9f, 0.5f);
            case "loamy_soil" -> new SoilNutrientData(0.8f, 0.8f, 0.8f, 0.7f, 0.7f, 1.2f);
            case "silty_soil" -> new SoilNutrientData(0.6f, 0.5f, 0.5f, 0.8f, 0.4f, 1.0f);
            case "peaty_soil" -> new SoilNutrientData(0.9f, 0.2f, 0.2f, 0.95f, 0.2f, 1.1f);
            case "chalky_soil" -> new SoilNutrientData(0.3f, 0.6f, 0.4f, 0.3f, 0.8f, 0.6f);
            case "saline_soil" -> new SoilNutrientData(0.1f, 0.1f, 0.1f, 0.5f, 0.5f, 0.1f);
            case "volcanic_ash_soil" -> new SoilNutrientData(0.9f, 0.9f, 0.9f, 0.6f, 0.6f, 1.5f);
            case "permafrost_soil" -> new SoilNutrientData(0.1f, 0.1f, 0.1f, 1.0f, 0.0f, 0.0f);
            case "laterite_soil" -> new SoilNutrientData(0.2f, 0.8f, 0.1f, 0.4f, 0.6f, 0.4f);
            default -> new SoilNutrientData(0.5f, 0.5f, 0.5f, 0.5f, 0.5f, 1.0f);
        };
    }
}

package com.horizonpack.horizonbiome.ai;

import net.minecraft.world.entity.Mob;
import net.minecraft.world.entity.LivingEntity;
import net.minecraft.world.entity.ai.goal.Goal;
import net.minecraft.world.phys.AABB;

import java.util.List;

public class PredatorHuntGoal extends Goal {
    private final Mob predator;
    private final double speedModifier;
    private LivingEntity target;

    public PredatorHuntGoal(Mob predator, double speedModifier) {
        this.predator = predator;
        this.speedModifier = speedModifier;
    }

    @Override
    public boolean canUse() {
        if (predator.getRandom().nextInt(100) != 0) return false;
        
        AABB searchArea = predator.getBoundingBox().inflate(16.0D);
        List<LivingEntity> prey = predator.level().getEntitiesOfClass(LivingEntity.class, searchArea, 
            e -> e.getType().is(com.horizonpack.horizonbiome.core.BiomeRegistries.PREY_TAG));
            
        if (!prey.isEmpty()) {
            this.target = prey.get(predator.getRandom().nextInt(prey.size()));
            return true;
        }
        return false;
    }

    @Override
    public void start() {
        predator.getNavigation().moveTo(target, speedModifier);
        predator.setTarget(target);
    }
}

package com.horizonpack.horizonbiome.entities;

import net.minecraft.nbt.CompoundTag;
import net.minecraft.network.syncher.EntityDataAccessor;
import net.minecraft.network.syncher.EntityDataSerializers;
import net.minecraft.network.syncher.SynchedEntityData;
import net.minecraft.world.entity.Entity;
import net.minecraft.world.entity.EntityType;
import net.minecraft.world.level.Level;

public class CarcassEntity extends Entity {
    
    // 0 = Fresh, 1 = Decomposing, 2 = Rotten, 3 = Skeleton
    private static final EntityDataAccessor<Integer> DECOMP_STAGE = 
        SynchedEntityData.defineId(CarcassEntity.class, EntityDataSerializers.INT);
        
    private int ageTicks = 0;
    public String sourceAnimalId = "";

    public CarcassEntity(EntityType<?> type, Level level) {
        super(type, level);
    }

    @Override
    protected void defineSynchedData(SynchedEntityData.Builder builder) {
        builder.define(DECOMP_STAGE, 0);
    }

    @Override
    public void tick() {
        super.tick();
        if (!level().isClientSide) {
            ageTicks++;
            // Stage advances every 48000 ticks (2 in-game days)
            int stage = Math.min(3, ageTicks / 48000);
            if (stage != this.entityData.get(DECOMP_STAGE)) {
                this.entityData.set(DECOMP_STAGE, stage);
            }
            
            // Attract scavengers if rotting
            if (stage >= 1 && ageTicks % 200 == 0) {
                com.horizonpack.horizonbiome.systems.DecompositionEngine.attractScavengers(this);
            }
            
            // Generate disease miasma if rotten
            if (stage == 2 && ageTicks % 100 == 0) {
                com.horizonpack.horizonbiome.systems.DecompositionEngine.emitDisease(this);
            }
        }
    }

    @Override
    protected void readAdditionalSaveData(CompoundTag tag) {
        ageTicks = tag.getInt("AgeTicks");
        sourceAnimalId = tag.getString("SourceAnimal");
        this.entityData.set(DECOMP_STAGE, tag.getInt("Stage"));
    }

    @Override
    protected void addAdditionalSaveData(CompoundTag tag) {
        tag.putInt("AgeTicks", ageTicks);
        tag.putString("SourceAnimal", sourceAnimalId);
        tag.putInt("Stage", this.entityData.get(DECOMP_STAGE));
    }
}

import net.neoforged.neoforge.event.tick.PlayerTickEvent;

@Mod.EventBusSubscriber(modid = HorizonBiome.MODID)
public class BiomeEventHandlers {

    @SubscribeEvent
    public static void onAnimalDeath(LivingDeathEvent event) {
        LivingEntity entity = event.getEntity();
        if (!entity.level().isClientSide && entity instanceof Animal) {
            // Cancel default drops, spawn carcass instead
            CarcassEntity carcass = new CarcassEntity(com.horizonpack.horizonbiome.core.BiomeRegistries.CARCASS.get(), entity.level());
            carcass.setPos(entity.getX(), entity.getY(), entity.getZ());
            carcass.sourceAnimalId = net.minecraft.core.registries.BuiltInRegistries.ENTITY_TYPE.getKey(entity.getType()).toString();
            entity.level().addFreshEntity(carcass);
            // We let the event proceed to kill the entity, but intercept loot logic elsewhere
        }
    }

    @SubscribeEvent
    public static void onPlayerTick(PlayerTickEvent.Post event) {
        if (!event.getEntity().level().isClientSide && event.getEntity().tickCount % 100 == 0) {
            DiseaseEngine.tickPlayerHealth((ServerPlayer) event.getEntity());
        }
    }
    
    @SubscribeEvent
    public static void onPlayerInteractEntity(PlayerInteractEvent.EntityInteract event) {
        if (event.getTarget() instanceof CarcassEntity carcass && !event.getLevel().isClientSide) {
            if (event.getItemStack().is(com.horizonpack.horizonbiome.core.BiomeRegistries.KNIVES_TAG)) {
                // Handle carving
                event.getLevel().addFreshEntity(new ItemEntity(event.getLevel(), carcass.getX(), carcass.getY(), carcass.getZ(), 
                    new net.minecraft.world.item.ItemStack(net.minecraft.world.item.Items.BEEF, 4)));
                carcass.discard();
            }
        }
    }
}

package com.horizonpack.horizonbiome.network.packets;

import com.horizonpack.horizonbiome.core.HorizonBiome;
import io.netty.buffer.ByteBuf;
import net.minecraft.network.codec.ByteBufCodecs;
import net.minecraft.network.codec.StreamCodec;
import net.minecraft.network.protocol.common.custom.CustomPacketPayload;
import net.minecraft.resources.ResourceLocation;
import net.neoforged.neoforge.network.handling.IPayloadContext;

import java.util.List;

public record SyncDiseasePacket(List<String> activeDiseases) implements CustomPacketPayload {

    public static final Type<SyncDiseasePacket> TYPE = new Type<>(ResourceLocation.fromNamespaceAndPath(HorizonBiome.MODID, "sync_diseases"));

    public static final StreamCodec<ByteBuf, SyncDiseasePacket> STREAM_CODEC = StreamCodec.composite(
        ByteBufCodecs.STRING_UTF8.apply(ByteBufCodecs.list()), SyncDiseasePacket::activeDiseases,
        SyncDiseasePacket::new
    );

    @Override
    public Type<SyncDiseasePacket> type() { return TYPE; }

    public static void handleClient(SyncDiseasePacket packet, IPayloadContext context) {
        context.enqueueWork(() -> {
            // Update client-side GUI state
            com.horizonpack.horizonbiome.client.gui.DiseaseHudOverlay.setActiveDiseases(packet.activeDiseases());
        });
    }
}

package com.horizonpack.horizonbiome.core;

import net.neoforged.neoforge.common.ModConfigSpec;

public class BiomeConfig {
    public static final ModConfigSpec COMMON_SPEC;
    public static final ModConfigSpec CLIENT_SPEC;
    
    public static class Common {
        public final ModConfigSpec.DoubleValue diseaseInfectionRate;
        public final ModConfigSpec.BooleanValue carcassDecomposition;
        public final ModConfigSpec.IntValue carcassRotTicks;
        public final ModConfigSpec.BooleanValue cropSeasonsStrict;
        
        Common(ModConfigSpec.Builder builder) {
            builder.push("microbiology");
            diseaseInfectionRate = builder.comment("Multiplier for infection chances").defineInRange("infectionRate", 1.0, 0.0, 5.0);
            builder.pop();
            
            builder.push("fauna");
            carcassDecomposition = builder.comment("Enable complex carcass rotting").define("carcassDecomposition", true);
            carcassRotTicks = builder.comment("Ticks until carcass rots completely").defineInRange("carcassRotTicks", 96000, 24000, 240000);
            builder.pop();
            
            builder.push("flora");
            cropSeasonsStrict = builder.comment("Crops die if planted in wrong season").define("cropSeasonsStrict", true);
            builder.pop();
        }
    }
    
    public static class Client {
        public final ModConfigSpec.BooleanValue showDiseaseHUD;
        
        Client(ModConfigSpec.Builder builder) {
            builder.push("gui");
            showDiseaseHUD = builder.comment("Show active diseases on HUD").define("showDiseaseHUD", true);
            builder.pop();
        }
    }
    
    public static final Common COMMON;
    public static final Client CLIENT;
    
    static {
        var commonPair = new ModConfigSpec.Builder().configure(Common::new);
        COMMON = commonPair.getLeft();
        COMMON_SPEC = commonPair.getRight();
        
        var clientPair = new ModConfigSpec.Builder().configure(Client::new);
        CLIENT = clientPair.getLeft();
        CLIENT_SPEC = clientPair.getRight();
    }
}

22. Critical Implementation Notes
Crop Seasonality & Growth Constraints:
Crops must evaluate horizonworld's SeasonData.getCurrentSeason() during randomTick. If a crop's favored growing season does not match the current season (e.g., trying to grow Tomatoes in WINTER), the crop halts growth. If cropSeasonsStrict is true in the config, the crop block state will transition directly to a dead/wilted state after 3 out-of-season ticks.
Soil Nutrient Depletion System:
HorizonCropBlock reads SoilNutrientData from the block directly beneath it. Upon successfully reaching maturity (age=7), the soil block's internal NBT (if a BlockEntity is used, or a variant block state if not) drops its N/P/K values. This enforces crop rotation or the application of fertilizers.
Disease Capability Persistence:
Active diseases must be serialized into the player's IHorizonPlayerData capability mapped under the "Diseases" compound tag. Do not use volatile variables. When the player dies, deficiency-based diseases clear, but contagious diseases persist across respawns if resetNutritionOnDeath is disabled in horizoncore's configuration.
Carcass Entity Network Sync & Performance:
The CarcassEntity does not have standard AI goals and does not move. Its DECOMP_STAGE is tracked via EntityDataAccessor to ensure all tracking clients immediately render the correct texture (Fresh -> Skeleton) without requiring custom packets. To preserve server tick rates after mass hunting, carcasses outside a 128-block radius of any player have their ticking paused until loaded again.
Predator/Prey AI Efficiency:
The PredatorHuntGoal uses getEntitiesOfClass combined with a tag check (BiomeRegistries.PREY_TAG). Doing this check every tick is incredibly expensive. Ensure canUse() has a built-in tick accumulator or random chance (if (predator.getRandom().nextInt(100) != 0) return false;) to stagger the AABB intersection checks across the server tick schedule.
Animal Breeding & Genetics NBT:
Animals bred by the player store inherited trait modifiers (Health, Speed, Attack) in their CompoundTag during addAdditionalSaveData. When a baby spawns, the BreedingHelper must average the parents' attributes, apply a slight Gaussian mutation, and push those values into the child's AttributeInstance to simulate realistic domestication over multiple generations.
Biome Modifier Spawning Safety:
When adding animals to biomes via the neoforge:add_spawns biome modifier, ensure you map them to the climate tags injected by horizonworld (e.g., #horizonworld:climate_tropical) rather than vanilla identifiers. This prevents spawns from breaking if horizonworld radically shifts the base terrain layout away from vanilla logic.
Climate Snapshot Caching for Flora/Fauna:
Do not call ClimateEngine.getSnapshot() from every crop's randomTick or every animal's tick method simultaneously. Animals must cache their current environmental temperature/humidity and only refresh it when traversing chunk boundaries, reducing overhead on horizonworld's cache map during dense entity moments.
