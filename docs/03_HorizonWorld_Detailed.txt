## 1. Overview & Responsibilities

HorizonWorld is the **planetary engine** of the Horizon Corebreach modpack. It replaces Minecraft's default world generation entirely and provides:

- **Continental Formation** — tectonic plate simulation producing distinct landmasses, ocean basins, mountain ranges, rift valleys, and island arcs; configurable presets (Pangaea, Earth-like, Archipelago, custom)
- **Geological Layer System** — accurate rock strata (sedimentary, igneous, metamorphic) per continent slab, with `GeologyProfile` objects that drive ore vein placement in HorizonData
- **Custom Chunk Generator** (`HorizonChunkGenerator`) — replaces `NoiseBasedChunkGenerator` with a multi-pass pipeline: continent mask → elevation noise → geological fill → surface layer → feature pass
- **Climate Zone System** — latitude-driven Koppen-style zones (Tropical, Arid, Temperate, Continental, Polar) applied per biome column; each zone defines temperature range, precipitation, humidity, and solar intensity
- **Weather Engine** — a per-dimension `WeatherState` machine that drives rain, snow, hail, fog, wind, thunderstorms, hurricanes, tornadoes, blizzards, monsoons, and dust storms; weather is spatially correlated across chunks (no per-chunk rain toggle)
- **Season System** — four seasons with configurable day lengths, temperature shifts, precipitation patterns, crop growth multipliers, and animal migration triggers; synced to in-game day count via HorizonCore's `DayAdvanceEvent`
- **Natural Disaster System** — earthquake, volcanic eruption, tsunami, flood, drought, wildfire, avalanche, landslide; each disaster has a geographic probability map, precondition checks, progression phases, and an aftermath cleanup phase
- **Ecosystem & Biome Ecology** — per-biome ecological balance tracker (predator/prey ratios, plant biomass, soil health, water table); imbalances trigger cascade events listened to by HorizonBiome
- **Planet SavedData** (`PlanetData`) — server-side singleton storing continent map, tectonic stress values, active disaster list, season state, and the global climate snapshot; never re-computed after world creation
- **World-specific blocks** — geological surface blocks (`LoamBlock`, `PeatBlock`, `CaicheBlock`, `PermafrostBlock`, `VolcanicAshBlock`, `TephriteBlock`), disaster aftermath blocks (`FloodSiltBlock`, `ScorchedEarthBlock`, `LandslideDebrisBlock`), and climate indicators (`FrostCrackBlock`, `DroughtCrackedEarthBlock`)

Every other Horizon mod that needs to know "where are we in the world" — biome type, latitude, climate zone, soil type, elevation — queries HorizonWorld's APIs.

## 3. Mod Entry Point

```java
package com.horizonpack.horizonworld.core;

import com.horizonpack.horizonworld.events.WorldEventHandlers;
import com.horizonpack.horizonworld.network.WorldPacketHandler;
import com.horizonpack.horizonworld.systems.*;
import net.neoforged.bus.api.IEventBus;
import net.neoforged.fml.ModContainer;
import net.neoforged.fml.common.Mod;
import net.neoforged.fml.config.ModConfig;
import net.neoforged.fml.event.lifecycle.FMLCommonSetupEvent;
import net.neoforged.neoforge.common.NeoForge;

@Mod(HorizonWorld.MODID)
public class HorizonWorld {
    public static final String MODID = "horizonworld";

    public HorizonWorld(IEventBus modEventBus, ModContainer modContainer) {
        // Config
        modContainer.registerConfig(ModConfig.Type.COMMON, WorldConfig.COMMON_SPEC);
        modContainer.registerConfig(ModConfig.Type.CLIENT, WorldConfig.CLIENT_SPEC);

        // All deferred registers
        WorldRegistries.BLOCKS.register(modEventBus);
        WorldRegistries.ITEMS.register(modEventBus);
        WorldRegistries.BLOCK_ENTITIES.register(modEventBus);
        WorldRegistries.BIOMES.register(modEventBus);
        WorldRegistries.CHUNK_GENERATORS.register(modEventBus);
        WorldRegistries.FEATURES.register(modEventBus);
        WorldRegistries.PARTICLE_TYPES.register(modEventBus);
        WorldRegistries.SOUND_EVENTS.register(modEventBus);
        WorldRegistries.CREATIVE_TABS.register(modEventBus);

        // Lifecycle
        modEventBus.addListener(this::commonSetup);
        modEventBus.addListener(WorldPacketHandler::register);

        // Forge event bus (game events)
        NeoForge.EVENT_BUS.register(WorldEventHandlers.class);
    }

    private void commonSetup(FMLCommonSetupEvent event) {
        event.enqueueWork(() -> {
            // Register chunk generator codec
            WorldRegistries.registerChunkGeneratorCodec();
            // Prime the climate cache
            ClimateEngine.init();
        });
    }
}
```

```java
package com.horizonpack.horizonworld.data.planet;

import com.mojang.serialization.Codec;
import com.mojang.serialization.MapCodec;
import com.mojang.serialization.codecs.RecordCodecBuilder;
import net.minecraft.resources.ResourceLocation;

import java.util.List;

/**
 * Immutable data object describing one landmass.
 * Registered in a DeferredRegister so data packs can define new continents.
 */
public record ContinentDefinition(
        ResourceLocation id,
        String displayName,
        TectonicType tectonicType,       // CONTINENTAL, OCEANIC, ISLAND_ARC, CRATON
        ResourceLocation geologyProfileId,
        float baseElevation,             // metres above sea level, 0–6000
        float seismicActivity,           // 0.0–1.0
        float volcanicActivity,          // 0.0–1.0
        List<ResourceBias> resourceBiases // which ores are more common
) {
    public static final MapCodec<ContinentDefinition> CODEC = RecordCodecBuilder.mapCodec(inst -> inst.group(
            ResourceLocation.CODEC.fieldOf("id").forGetter(ContinentDefinition::id),
            Codec.STRING.fieldOf("display_name").forGetter(ContinentDefinition::displayName),
            TectonicType.CODEC.fieldOf("tectonic_type").forGetter(ContinentDefinition::tectonicType),
            ResourceLocation.CODEC.fieldOf("geology_profile").forGetter(ContinentDefinition::geologyProfileId),
            Codec.FLOAT.fieldOf("base_elevation").forGetter(ContinentDefinition::baseElevation),
            Codec.FLOAT.fieldOf("seismic_activity").forGetter(ContinentDefinition::seismicActivity),
            Codec.FLOAT.fieldOf("volcanic_activity").forGetter(ContinentDefinition::volcanicActivity),
            ResourceBias.CODEC.listOf().fieldOf("resource_biases").forGetter(ContinentDefinition::resourceBiases)
    ).apply(inst, ContinentDefinition::new));

    public enum TectonicType {
        CONTINENTAL, OCEANIC, ISLAND_ARC, CRATON;
        public static final Codec<TectonicType> CODEC = Codec.STRING.xmap(
                TectonicType::valueOf, TectonicType::name);
    }

    public record ResourceBias(ResourceLocation oreId, float multiplier) {
        public static final MapCodec<ResourceBias> CODEC = RecordCodecBuilder.mapCodec(inst -> inst.group(
                ResourceLocation.CODEC.fieldOf("ore").forGetter(ResourceBias::oreId),
                Codec.FLOAT.fieldOf("multiplier").forGetter(ResourceBias::multiplier)
        ).apply(inst, ResourceBias::new));
    }
}
```

### 4.2 `TectonicPlate.java`

```java
package com.horizonpack.horizonworld.data.planet;

import net.minecraft.nbt.CompoundTag;
import net.minecraft.nbt.ListTag;
import net.minecraft.nbt.Tag;

import java.util.ArrayList;
import java.util.List;

/**
 * Runtime tectonic plate.  Created once during world-gen init and stored in PlanetData.
 * Tracks boundary stress that drives earthquake/volcanic disaster probability.
 */
public class TectonicPlate {
    private final int plateId;
    private final List<PlateBoundary> boundaries = new ArrayList<>();
    /** 0.0 = calm; 1.0 = imminent rupture */
    private float stress;
    private int daysSinceLastQuake;

    public TectonicPlate(int plateId) {
        this.plateId = plateId;
        this.stress = 0f;
    }

    /** Called by TectonicEngine every in-game day. */
    public void accumulateStress(float dailyRate) {
        stress = Math.min(1.0f, stress + dailyRate);
        daysSinceLastQuake++;
    }

    public void releaseStress(float amount) {
        stress = Math.max(0f, stress - amount);
        daysSinceLastQuake = 0;
    }

    public float stress()            { return stress; }
    public int plateId()             { return plateId; }
    public List<PlateBoundary> boundaries() { return boundaries; }

    // --- NBT ---
    public CompoundTag save() {
        CompoundTag tag = new CompoundTag();
        tag.putInt("PlateId", plateId);
        tag.putFloat("Stress", stress);
        tag.putInt("DaysSinceQuake", daysSinceLastQuake);
        ListTag bList = new ListTag();
        for (PlateBoundary b : boundaries) bList.add(b.save());
        tag.put("Boundaries", bList);
        return tag;
    }

    public static TectonicPlate load(CompoundTag tag) {
        TectonicPlate plate = new TectonicPlate(tag.getInt("PlateId"));
        plate.stress = tag.getFloat("Stress");
        plate.daysSinceLastQuake = tag.getInt("DaysSinceQuake");
        ListTag bList = tag.getList("Boundaries", Tag.TAG_COMPOUND);
        for (int i = 0; i < bList.size(); i++) {
            plate.boundaries.add(PlateBoundary.load(bList.getCompound(i)));
        }
        return plate;
    }

    public record PlateBoundary(int otherPlateId, BoundaryType type, int[] epicenterX, int[] epicenterZ) {
        public enum BoundaryType { CONVERGENT, DIVERGENT, TRANSFORM }

        public CompoundTag save() {
            CompoundTag tag = new CompoundTag();
            tag.putInt("OtherPlate", otherPlateId);
            tag.putString("BoundaryType", type.name());
            tag.putIntArray("EpicenterX", epicenterX);
            tag.putIntArray("EpicenterZ", epicenterZ);
            return tag;
        }

        public static PlateBoundary load(CompoundTag tag) {
            return new PlateBoundary(
                    tag.getInt("OtherPlate"),
                    BoundaryType.valueOf(tag.getString("BoundaryType")),
                    tag.getIntArray("EpicenterX"),
                    tag.getIntArray("EpicenterZ"));
        }
    }
}
```

```java
package com.horizonpack.horizonworld.data.planet;

import com.mojang.serialization.Codec;
import com.mojang.serialization.MapCodec;
import com.mojang.serialization.codecs.RecordCodecBuilder;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.world.level.block.state.BlockState;

import java.util.List;

/**
 * Describes the rock strata that GeologyFillPass uses to fill chunk columns.
 * Data-pack driven via CODEC.
 */
public record GeologyProfile(
        ResourceLocation id,
        List<StrataLayer> layers,         // top→bottom order; last covers to bedrock
        float metamorphicDepthFraction,   // fraction of world height where metamorphic rock starts
        float igneousIntrusions           // probability per chunk of igneous intrusion column
) {
    public static final MapCodec<GeologyProfile> CODEC = RecordCodecBuilder.mapCodec(inst -> inst.group(
            ResourceLocation.CODEC.fieldOf("id").forGetter(GeologyProfile::id),
            StrataLayer.CODEC.listOf().fieldOf("layers").forGetter(GeologyProfile::layers),
            Codec.FLOAT.fieldOf("metamorphic_depth_fraction").forGetter(GeologyProfile::metamorphicDepthFraction),
            Codec.FLOAT.fieldOf("igneous_intrusions").forGetter(GeologyProfile::igneousIntrusions)
    ).apply(inst, GeologyProfile::new));

    public record StrataLayer(
            ResourceLocation blockId,   // horizondata block id
            int minThickness,           // in blocks
            int maxThickness,
            float noiseVariation        // how much thickness varies per chunk
    ) {
        public static final MapCodec<StrataLayer> CODEC = RecordCodecBuilder.mapCodec(inst -> inst.group(
                ResourceLocation.CODEC.fieldOf("block").forGetter(StrataLayer::blockId),
                Codec.INT.fieldOf("min_thickness").forGetter(StrataLayer::minThickness),
                Codec.INT.fieldOf("max_thickness").forGetter(StrataLayer::maxThickness),
                Codec.FLOAT.fieldOf("noise_variation").forGetter(StrataLayer::noiseVariation)
        ).apply(inst, StrataLayer::new));
    }
}
```

```java
        setDirty();
    }

    public void removeDisaster(DisasterInstance d) {
        activeDisasters.remove(d);
        setDirty();
    }

    public GeologyProfile getProfile(String continentId) {
        return continentProfiles.get(continentId);
    }

    public void putProfile(String continentId, GeologyProfile profile) {
        continentProfiles.put(continentId, profile);
        setDirty();
    }

    // --- Serialization ---
    @Override
    public CompoundTag save(CompoundTag tag) {
        if (continentMap != null) tag.put("ContinentMap", continentMap.save());
        ListTag plateList = new ListTag();
        for (TectonicPlate p : plates) plateList.add(p.save());
        tag.put("Plates", plateList);
        ListTag disasterList = new ListTag();
        for (DisasterInstance d : activeDisasters) disasterList.add(d.save());
        tag.put("ActiveDisasters", disasterList);
        return tag;
    }

    private static PlanetData load(CompoundTag tag) {
        PlanetData data = new PlanetData();
        if (tag.contains("ContinentMap")) {
            data.continentMap = ContinentMap.load(tag.getCompound("ContinentMap"));
        }
        ListTag plateList = tag.getList("Plates", Tag.TAG_COMPOUND);
        for (int i = 0; i < plateList.size(); i++) {
            data.plates.add(TectonicPlate.load(plateList.getCompound(i)));
        }
        ListTag disasterList = tag.getList("ActiveDisasters", Tag.TAG_COMPOUND);
        for (int i = 0; i < disasterList.size(); i++) {
            data.activeDisasters.add(DisasterInstance.load(disasterList.getCompound(i)));
        }
        return data;
    }
```

```java
package com.horizonpack.horizonworld.data.planet;

import net.minecraft.nbt.CompoundTag;
import net.minecraft.nbt.IntArrayTag;

/**
 * A 2D grid mapping (worldX, worldZ) → continentId (byte, -1 = ocean).
 * Grid resolution: 1 cell = CELL_SIZE blocks (default 64).
 */
public class ContinentMap {
    public static final int CELL_SIZE = 64;

    private final int gridWidth;
    private final int gridHeight;
    private final byte[] cells; // -1=ocean, 0-126=continent index

    public ContinentMap(int gridWidth, int gridHeight) {
        this.gridWidth = gridWidth;
        this.gridHeight = gridHeight;
        this.cells = new byte[gridWidth * gridHeight];
        java.util.Arrays.fill(cells, (byte) -1);
    }

    public byte getContinentAt(int blockX, int blockZ) {
        int gx = Math.floorDiv(blockX, CELL_SIZE);
        int gz = Math.floorDiv(blockZ, CELL_SIZE);
        if (gx < 0 || gz < 0 || gx >= gridWidth || gz >= gridHeight) return -1;
        return cells[gz * gridWidth + gx];
    }

    public boolean isOcean(int blockX, int blockZ) {
        return getContinentAt(blockX, blockZ) == -1;
    }

    public void setContinent(int gx, int gz, byte id) {
        if (gx < 0 || gz < 0 || gx >= gridWidth || gz >= gridHeight) return;
        cells[gz * gridWidth + gx] = id;
    }

    public CompoundTag save() {
        CompoundTag tag = new CompoundTag();
        tag.putInt("GridW", gridWidth);
        tag.putInt("GridH", gridHeight);
        int[] intCells = new int[cells.length];
        for (int i = 0; i < cells.length; i++) intCells[i] = cells[i];
        tag.put("Cells", new IntArrayTag(intCells));
        return tag;
    }

    public static ContinentMap load(CompoundTag tag) {
        int w = tag.getInt("GridW");
        int h = tag.getInt("GridH");
        ContinentMap map = new ContinentMap(w, h);
        int[] intCells = tag.getIntArray("Cells");
        for (int i = 0; i < intCells.length && i < map.cells.length; i++) {
            map.cells[i] = (byte) intCells[i];
        }
        return map;
    }
}
```

```java
    }
```

### 6.2 `ElevationPass.java`

```java
package com.horizonpack.horizonworld.worldgen.passes;

import com.horizonpack.horizonworld.worldgen.HorizonChunkGenerator.WorldGenSettings;
import com.horizonpack.horizonworld.worldgen.noise.ElevationNoise;
import net.minecraft.world.level.chunk.ChunkAccess;
import net.minecraft.world.level.levelgen.RandomState;

public class ElevationPass {
    private final WorldGenSettings settings;
    private final ElevationNoise noise;

    public ElevationPass(WorldGenSettings settings) {
        this.settings = settings;
        this.noise = new ElevationNoise(settings.continentSeed());
    }

    /** Returns a 16×16 elevation array (block height) for the given chunk. */
    public int[][] compute(ChunkAccess chunk, RandomState random) {
        int[][] map = new int[16][16];
        int baseX = chunk.getPos().getMinBlockX();
        int baseZ = chunk.getPos().getMinBlockZ();
        for (int dx = 0; dx < 16; dx++) {
            for (int dz = 0; dz < 16; dz++) {
                map[dx][dz] = sampleElevation(baseX + dx, baseZ + dz);
            }
        }
        return map;
    }

    public int getBaseHeight(int x, int z, RandomState random) {
        return sampleElevation(x, z);
    }

    private int sampleElevation(int x, int z) {
        double raw = noise.sample(x, z); // −1.0 → 1.0
        // Map to block height: sea level ± half world height
        int seaLevel = settings.seaLevel();
        int halfHeight = settings.worldHeight() / 2;
        return (int) (seaLevel + raw * halfHeight);
    }
}
```

### 6.3 `GeologyFillPass.java`

```java
package com.horizonpack.horizonworld.worldgen.passes;

import com.horizonpack.horizonworld.core.HorizonWorld;
import com.horizonpack.horizonworld.data.planet.GeologyProfile;
import com.horizonpack.horizonworld.data.planet.PlanetData;
import com.horizonpack.horizonworld.worldgen.HorizonChunkGenerator.WorldGenSettings;
import net.minecraft.core.BlockPos;
import net.minecraft.core.registries.BuiltInRegistries;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.world.level.block.Blocks;
import net.minecraft.world.level.block.state.BlockState;
import net.minecraft.world.level.chunk.ChunkAccess;
import net.minecraft.world.level.levelgen.RandomState;

public class GeologyFillPass {
    private final WorldGenSettings settings;

    public GeologyFillPass(WorldGenSettings settings) { this.settings = settings; }

    public void fill(ChunkAccess chunk, int[][] elevationMap, boolean[][] oceanMask, RandomState random) {
        int baseX = chunk.getPos().getMinBlockX();
        int baseZ = chunk.getPos().getMinBlockZ();
        int minY  = chunk.getMinBuildHeight();

        for (int dx = 0; dx < 16; dx++) {
            for (int dz = 0; dz < 16; dz++) {
                int worldX = baseX + dx;
                int worldZ = baseZ + dz;
                int surfaceY = elevationMap[dx][dz];
                boolean ocean = oceanMask[dx][dz];

                // Fill bedrock at bottom
                chunk.setBlockState(new BlockPos(worldX, minY, worldZ), Blocks.BEDROCK.defaultBlockState(), false);

                if (ocean) {
                    fillOceanColumn(chunk, worldX, worldZ, surfaceY, minY + 1);
                } else {
                    // TODO: look up GeologyProfile from PlanetData and use strata layers
                    fillDefaultLandColumn(chunk, worldX, worldZ, surfaceY, minY + 1, random);
                }
            }
        }
    }

    private void fillOceanColumn(ChunkAccess chunk, int x, int z, int seaBed, int minY) {
        for (int y = minY; y <= seaBed; y++) {
            chunk.setBlockState(new BlockPos(x, y, z),
                    y < seaBed - 3 ? Blocks.STONE.defaultBlockState() : Blocks.GRAVEL.defaultBlockState(), false);
        }
        // Fill water above seabed up to sea level
        for (int y = seaBed + 1; y <= settings.seaLevel(); y++) {
            chunk.setBlockState(new BlockPos(x, y, z), Blocks.WATER.defaultBlockState(), false);
        }
    }

    private void fillDefaultLandColumn(ChunkAccess chunk, int x, int z, int surfaceY, int minY, RandomState random) {
        for (int y = minY; y <= surfaceY; y++) {
            double depthFraction = (double)(surfaceY - y) / (surfaceY - minY);
            BlockState state;
            if (depthFraction > 0.8) {
                state = Blocks.DEEPSLATE.defaultBlockState();
            } else if (depthFraction > 0.5) {
                state = Blocks.STONE.defaultBlockState();
            } else if (depthFraction > 0.15) {
                state = Blocks.STONE.defaultBlockState(); // replaced by geology profile in production
            } else {
                state = Blocks.STONE.defaultBlockState();
            }
            chunk.setBlockState(new BlockPos(x, y, z), state, false);
        }
    }

    public BlockState getSampleBlock(int x, int y, int z, RandomState random) {
        return Blocks.STONE.defaultBlockState();
    }
}
```

```java
package com.horizonpack.horizonworld.data.climate;

import com.mojang.serialization.Codec;

/**
 * Koppen-derived climate classification used for biome assignment
 * and weather probability weights.
 */
public enum ClimateZone {
    /** 0–23° latitude: high temp, high rainfall year-round */
    TROPICAL(28f, 32f, 0.8f, 0.9f),
    /** Desert/steppe: low rainfall, high temp variation */
    ARID(15f, 40f, 0.05f, 0.2f),
    /** Moderate temp and rainfall, four seasons */
    TEMPERATE(5f, 25f, 0.4f, 0.6f),
    /** Cold winters, warm summers, moderate rain */
    CONTINENTAL(-10f, 22f, 0.35f, 0.55f),
    /** Near poles: extreme cold, low precipitation */
    POLAR(-40f, 0f, 0.1f, 0.3f);

    public final float minTemp;   // °C minimum seasonal
    public final float maxTemp;   // °C maximum seasonal
    public final float minPrecip; // fraction of days with precipitation (low season)
    public final float maxPrecip; // fraction of days with precipitation (high season)

    ClimateZone(float minTemp, float maxTemp, float minPrecip, float maxPrecip) {
        this.minTemp   = minTemp;
        this.maxTemp   = maxTemp;
        this.minPrecip = minPrecip;
        this.maxPrecip = maxPrecip;
    }

    public static final Codec<ClimateZone> CODEC =
        Codec.STRING.xmap(ClimateZone::valueOf, ClimateZone::name);

    /** Latitude in degrees (0=equator, 90=pole) → zone */
    public static ClimateZone fromLatitude(float latitude) {
        float abs = Math.abs(latitude);
        if (abs < 23f)  return TROPICAL;
        if (abs < 35f)  return ARID;
        if (abs < 55f)  return TEMPERATE;
        if (abs < 70f)  return CONTINENTAL;
        return POLAR;
    }
}
```

### 7.3 `ClimateEngine.java`

```java
package com.horizonpack.horizonworld.systems;

import com.horizonpack.horizonworld.data.climate.ClimateSnapshot;
import com.horizonpack.horizonworld.data.climate.ClimateZone;
import com.horizonpack.horizonworld.data.climate.Season;
import com.horizonpack.horizonworld.utils.ClimateCache;
import com.horizonpack.horizonworld.utils.LatLonUtil;
import net.minecraft.core.BlockPos;
import net.minecraft.server.level.ServerLevel;

public final class ClimateEngine {
    private ClimateEngine() {}

    private static ClimateCache cache;

    public static void init() {
        cache = new ClimateCache(4096); // LRU of 4096 chunk entries
    }

    /**
     * Primary API called by other mods.
     * Thread-safe: cache is synchronized; computation is pure.
     */
    public static ClimateSnapshot getSnapshot(ServerLevel level, BlockPos pos) {
        long key = (long)(pos.getX() >> 4) << 32 | (pos.getZ() >> 4 & 0xFFFFFFFFL);
        ClimateSnapshot cached = cache.get(key);
        if (cached != null) return cached;

        ClimateSnapshot computed = compute(level, pos);
        cache.put(key, computed);
        return computed;
    }

    /** Invalidate cache for a chunk (called on season change or disaster). */
    public static void invalidateChunk(int chunkX, int chunkZ) {
        long key = (long)chunkX << 32 | (chunkZ & 0xFFFFFFFFL);
        cache.remove(key);
    }

    /** Invalidate all cached snapshots (season transition). */
    public static void invalidateAll() {
        cache.clear();
    }

    private static ClimateSnapshot compute(ServerLevel level, BlockPos pos) {
        float[] latLon   = LatLonUtil.toLatLon(level, pos);
        ClimateZone zone = ClimateZone.fromLatitude(latLon[0]);
        Season season    = SeasonEngine.getCurrentSeason(level);

        float tempBase   = (zone.minTemp + zone.maxTemp) / 2f;
        float seasonOffset = switch (season) {
            case SUMMER -> (zone.maxTemp - tempBase) * 0.8f;
            case WINTER -> (zone.minTemp - tempBase) * 0.8f;
            default     -> 0f;
        };
        float temp = tempBase + seasonOffset;

        // Altitude cooling: −6.5°C per 1000m
        int elevMetres = Math.max(0, pos.getY() - 63) * 4; // approximate
        temp -= (elevMetres / 1000f) * 6.5f;

        float precipFraction = (zone.minPrecip + zone.maxPrecip) / 2f;
        float humidity = precipFraction * 1.2f;

        return new ClimateSnapshot(zone, temp, Math.min(1f, humidity),
                precipFraction, 20f, 0.8f, elevMetres);
    }
}
```

```java
package com.horizonpack.horizonworld.data.climate;

import net.minecraft.nbt.CompoundTag;

/**
 * Mutable weather state for one dimension.
 * Stored in SeasonData; updated by WeatherEngine every in-game hour.
 */
public class WeatherState {
    private WeatherType current = WeatherType.CLEAR;
    private WeatherType queued  = null;
    private int ticksRemaining  = 0;       // ticks at current weather
    private int transitionTicks = 0;       // ticks until queued takes effect
    private float intensity     = 1.0f;    // 0.0–1.0 modifier on particle density/damage

    public WeatherType getCurrent()  { return current; }
    public float getIntensity()      { return intensity; }
    public boolean isTransitioning() { return transitionTicks > 0; }

    public void tick() {
        if (transitionTicks > 0) {
            transitionTicks--;
            if (transitionTicks == 0 && queued != null) {
                current = queued;
                queued  = null;
            }
        } else if (ticksRemaining > 0) {
            ticksRemaining--;
        }
        // WeatherEngine.tick() replaces when ticksRemaining hits 0
    }

    public boolean needsNewWeather() { return ticksRemaining <= 0 && !isTransitioning(); }

    public void setWeather(WeatherType type, int durationTicks, int transitionTicks, float intensity) {
        this.queued           = type;
        this.transitionTicks  = transitionTicks;
        this.ticksRemaining   = durationTicks;
        this.intensity        = intensity;
    }

    // --- NBT ---
    public void save(CompoundTag tag) {
        tag.putString("Current", current.name());
        if (queued != null) tag.putString("Queued", queued.name());
        tag.putInt("TicksRemaining", ticksRemaining);
        tag.putInt("TransitionTicks", transitionTicks);
        tag.putFloat("Intensity", intensity);
    }

    public void load(CompoundTag tag) {
        current         = WeatherType.valueOf(tag.getString("Current"));
        queued          = tag.contains("Queued") ? WeatherType.valueOf(tag.getString("Queued")) : null;
        ticksRemaining  = tag.getInt("TicksRemaining");
        transitionTicks = tag.getInt("TransitionTicks");
        intensity       = tag.getFloat("Intensity");
    }
}
```

### 8.3 `WeatherEngine.java`

```java
package com.horizonpack.horizonworld.systems;

import com.horizonpack.horizonworld.data.climate.*;
import com.horizonpack.horizonworld.events.custom.WeatherChangeEvent;
import net.minecraft.server.level.ServerLevel;
import net.neoforged.neoforge.common.NeoForge;

import java.util.Random;

public final class WeatherEngine {
    private WeatherEngine() {}

    /**
     * Called every server tick by WorldEventHandlers.
     * Only processes every 20 ticks (once per second) for performance.
     */
    private static int tickAccum = 0;

    public static void tick(ServerLevel level, WeatherState state) {
        state.tick();
        tickAccum++;
        if (tickAccum < 20) return;
        tickAccum = 0;

        if (state.needsNewWeather()) {
            WeatherType next = selectNextWeather(level, state);
            int duration    = computeDuration(next);
            int transition  = 40; // 2 seconds cross-fade
            float intensity = computeIntensity(level, next);

            WeatherType prev = state.getCurrent();
            state.setWeather(next, duration, transition, intensity);

            if (prev != next) {
                NeoForge.EVENT_BUS.post(new WeatherChangeEvent(level, prev, next));
            }
        }
    }

    private static WeatherType selectNextWeather(ServerLevel level, WeatherState current) {
        // Get climate at world origin as a representative sample
        ClimateSnapshot snap = ClimateEngine.getSnapshot(level, net.minecraft.core.BlockPos.ZERO);
        Season season = SeasonEngine.getCurrentSeason(level);
        Random rand = new Random(level.getGameTime());

        return switch (snap.zone()) {
            case TROPICAL  -> selectTropical(season, rand, snap);
            case ARID      -> selectArid(season, rand, snap);
            case TEMPERATE -> selectTemperate(season, rand, snap);
            case CONTINENTAL -> selectContinental(season, rand, snap);
            case POLAR     -> selectPolar(season, rand, snap);
        };
    }

    private static WeatherType selectTropical(Season s, Random r, ClimateSnapshot snap) {
        float roll = r.nextFloat();
        if (s == Season.SUMMER || s == Season.AUTUMN) {
            if (roll < 0.05f) return WeatherType.HURRICANE;
            if (roll < 0.25f) return WeatherType.MONSOON;
            if (roll < 0.55f) return WeatherType.HEAVY_RAIN;
            if (roll < 0.70f) return WeatherType.THUNDERSTORM;
            return WeatherType.CLEAR;
        } else {
            if (roll < 0.35f) return WeatherType.LIGHT_RAIN;
            if (roll < 0.50f) return WeatherType.HEAVY_RAIN;
            return WeatherType.CLEAR;
        }
    }

    private static WeatherType selectArid(Season s, Random r, ClimateSnapshot snap) {
        float roll = r.nextFloat();
        if (roll < 0.10f) return WeatherType.DUST_STORM;
        if (roll < 0.15f) return WeatherType.LIGHT_RAIN;
        if (roll < 0.17f) return WeatherType.THUNDERSTORM;
        return WeatherType.CLEAR;
    }

    private static WeatherType selectTemperate(Season s, Random r, ClimateSnapshot snap) {
        float roll = r.nextFloat();
        return switch (s) {
            case SPRING -> roll < 0.40f ? WeatherType.LIGHT_RAIN
                         : roll < 0.55f ? WeatherType.HEAVY_RAIN
                         : roll < 0.60f ? WeatherType.THUNDERSTORM
                         : WeatherType.CLEAR;
            case SUMMER -> roll < 0.20f ? WeatherType.LIGHT_RAIN
                         : roll < 0.30f ? WeatherType.THUNDERSTORM
                         : roll < 0.32f ? WeatherType.TORNADO
                         : WeatherType.CLEAR;
            case AUTUMN -> roll < 0.45f ? WeatherType.LIGHT_RAIN
                         : roll < 0.55f ? WeatherType.FOG
                         : WeatherType.CLEAR;
            case WINTER -> roll < 0.35f ? WeatherType.SNOW
                         : roll < 0.45f ? WeatherType.BLIZZARD
                         : roll < 0.50f ? WeatherType.FOG
                         : WeatherType.CLEAR;
        };
    }

    private static WeatherType selectContinental(Season s, Random r, ClimateSnapshot snap) {
        float roll = r.nextFloat();
        return switch (s) {
            case WINTER -> roll < 0.50f ? WeatherType.BLIZZARD
                         : roll < 0.65f ? WeatherType.SNOW
                         : WeatherType.CLEAR;
            case SUMMER -> roll < 0.25f ? WeatherType.HEAVY_RAIN
                         : roll < 0.35f ? WeatherType.THUNDERSTORM
                         : WeatherType.CLEAR;
            default     -> roll < 0.30f ? WeatherType.LIGHT_RAIN : WeatherType.CLEAR;
        };
    }

    private static WeatherType selectPolar(Season s, Random r, ClimateSnapshot snap) {
        float roll = r.nextFloat();
        return roll < 0.60f ? WeatherType.BLIZZARD
             : roll < 0.75f ? WeatherType.SNOW
             : WeatherType.CLEAR;
    }

    private static int computeDuration(WeatherType type) {
        return switch (type) {
            case CLEAR      -> 6000 + (int)(Math.random() * 12000); // 5–15 min
            case LIGHT_RAIN -> 2400 + (int)(Math.random() * 4800);
            case HEAVY_RAIN, MONSOON -> 3600 + (int)(Math.random() * 3600);
            case THUNDERSTORM -> 1200 + (int)(Math.random() * 2400);
            case SNOW, FOG   -> 4800 + (int)(Math.random() * 9600);
            case BLIZZARD    -> 6000 + (int)(Math.random() * 6000);
            case HURRICANE, TORNADO -> 600 + (int)(Math.random() * 1200);
            case HAIL        -> 400  + (int)(Math.random() * 800);
            case DUST_STORM  -> 2400 + (int)(Math.random() * 4800);
            case VOLCANIC_ASH_FALL -> 12000; // driven by disaster, not duration
        };
    }

    private static float computeIntensity(ServerLevel level, WeatherType type) {
        return 0.5f + (float)Math.random() * 0.5f;
    }
}
```

### 10.4 `DisasterEngine.java`

```java
package com.horizonpack.horizonworld.systems;

import com.horizonpack.horizonworld.core.WorldConfig;
import com.horizonpack.horizonworld.data.disaster.*;
import com.horizonpack.horizonworld.data.planet.PlanetData;
import com.horizonpack.horizonworld.data.planet.TectonicPlate;
import com.horizonpack.horizonworld.events.custom.*;
import net.minecraft.core.BlockPos;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.world.level.block.Blocks;
import net.neoforged.neoforge.common.NeoForge;

import java.util.ArrayList;
import java.util.List;
import java.util.Random;

public final class DisasterEngine {
    private DisasterEngine() {}

    // Ticks per disaster check (every 10 seconds)
    private static final int CHECK_INTERVAL = 200;
    private static int tickAccum = 0;

    public static void tick(ServerLevel level) {
        PlanetData planet = PlanetData.get(level);

        // Tick existing disasters
        List<DisasterInstance> toRemove = new ArrayList<>();
        for (DisasterInstance disaster : planet.activeDisasters()) {
            disaster.tick();
            tickDisaster(level, disaster, planet);
            if (disaster.isFinished()) toRemove.add(disaster);
        }
        toRemove.forEach(planet::removeDisaster);

        // Check for new disasters
        tickAccum++;
        if (tickAccum >= CHECK_INTERVAL) {
            tickAccum = 0;
            maybeSpawnDisaster(level, planet);
        }
    }

    private static void tickDisaster(ServerLevel level, DisasterInstance d, PlanetData planet) {
        int phaseDuration = getPhaseDuration(d);
        if (d.ticksInPhase() >= phaseDuration) {
            DisasterPhase prev = d.phase();
            d.advancePhase();
            NeoForge.EVENT_BUS.post(new DisasterPhaseEvent(level, d, prev, d.phase()));

            if (d.phase() == DisasterPhase.FINISHED) {
                NeoForge.EVENT_BUS.post(new DisasterEndEvent(level, d));
            } else if (d.phase() == DisasterPhase.AFTERMATH) {
                applyAftermath(level, d);
            }
        }

        // Apply per-tick effects at PEAK
        if (d.phase() == DisasterPhase.PEAK) {
            applyPeakEffects(level, d);
        }
    }

    private static int getPhaseDuration(DisasterInstance d) {
        return switch (d.phase()) {
            case PRECURSOR -> 1200;  // 1 minute warning
            case ACTIVE    -> 600;
            case PEAK      -> 1200;
            case WANING    -> 600;
            case AFTERMATH -> 2400;
            default        -> 0;
        };
    }

    private static void applyPeakEffects(ServerLevel level, DisasterInstance d) {
        Random rand = new Random(level.getGameTime() + d.id().getMostSignificantBits());
        int radius = (int)(d.type().effectRadius * d.severity());

        switch (d.type()) {
            case EARTHQUAKE -> applyEarthquakeEffects(level, d.epicenter(), radius, rand);
            case VOLCANIC_ERUPTION -> applyVolcanicEffects(level, d.epicenter(), radius, rand);
            case WILDFIRE    -> applyWildfireEffects(level, d.epicenter(), radius, rand);
            case FLOOD       -> applyFloodEffects(level, d.epicenter(), radius, rand);
            default -> {} // Complex disasters handled in their own engine methods
        }
    }

    private static void applyEarthquakeEffects(ServerLevel level, BlockPos epicenter, int radius, Random rand) {
        // Sample a small number of blocks per tick to avoid lag
        for (int i = 0; i < 5; i++) {
            int dx = rand.nextInt(radius * 2) - radius;
            int dz = rand.nextInt(radius * 2) - radius;
            int dist = (int)Math.sqrt(dx*dx + dz*dz);
            if (dist > radius) continue;

            BlockPos surface = level.getHeightmapPos(
                net.minecraft.world.level.levelgen.Heightmap.Types.WORLD_SURFACE,
                epicenter.offset(dx, 0, dz));

            // Chance to collapse surface blocks
            float collapseProbability = 1f - ((float)dist / radius);
            if (rand.nextFloat() < collapseProbability * 0.05f) {
                if (!level.getBlockState(surface).isAir()) {
                    level.setBlock(surface, Blocks.AIR.defaultBlockState(), 3);
                    level.setBlock(surface.below(), Blocks.GRAVEL.defaultBlockState(), 3);
                }
            }
        }
    }

    private static void applyVolcanicEffects(ServerLevel level, BlockPos epicenter, int radius, Random rand) {
        for (int i = 0; i < 3; i++) {
            double angle  = rand.nextDouble() * Math.PI * 2;
            double dist   = rand.nextDouble() * radius;
            int tx = epicenter.getX() + (int)(Math.cos(angle) * dist);
            int tz = epicenter.getZ() + (int)(Math.sin(angle) * dist);
            BlockPos surface = level.getHeightmapPos(
                net.minecraft.world.level.levelgen.Heightmap.Types.WORLD_SURFACE,
                new BlockPos(tx, epicenter.getY(), tz));
            level.setBlock(surface, Blocks.MAGMA_BLOCK.defaultBlockState(), 3);
        }
    }

    private static void applyWildfireEffects(ServerLevel level, BlockPos epicenter, int radius, Random rand) {
        for (int i = 0; i < 8; i++) {
            int dx = rand.nextInt(radius * 2) - radius;
            int dz = rand.nextInt(radius * 2) - radius;
            BlockPos surface = level.getHeightmapPos(
                net.minecraft.world.level.levelgen.Heightmap.Types.WORLD_SURFACE,
                epicenter.offset(dx, 0, dz)).above();
            if (level.getBlockState(surface.below()).isFlammable(level, surface.below(), net.minecraft.core.Direction.UP)) {
                level.setBlock(surface, Blocks.FIRE.defaultBlockState(), 3);
            }
        }
    }

    private static void applyFloodEffects(ServerLevel level, BlockPos epicenter, int radius, Random rand) {
        for (int i = 0; i < 10; i++) {
            int dx = rand.nextInt(radius * 2) - radius;
            int dz = rand.nextInt(radius * 2) - radius;
            BlockPos surface = level.getHeightmapPos(
                net.minecraft.world.level.levelgen.Heightmap.Types.WORLD_SURFACE,
                epicenter.offset(dx, 0, dz));
            if (level.getBlockState(surface).isAir()) {
                level.setBlock(surface, Blocks.WATER.defaultBlockState(), 3);
            }
        }
    }

    private static void applyAftermath(ServerLevel level, DisasterInstance d) {
        // Place aftermath blocks: scorched earth, silt, debris
        // Implementation replaces PEAK-affected blocks with aftermath variants
    }

    private static void maybeSpawnDisaster(ServerLevel level, PlanetData planet) {
        if (!WorldConfig.COMMON.enableDisasters.get()) return;
        if (planet.activeDisasters().size() >= WorldConfig.COMMON.maxSimultaneousDisasters.get()) return;

        Random rand = new Random(level.getGameTime());
        float roll = rand.nextFloat();
        float freq = WorldConfig.COMMON.disasterFrequency.get().floatValue();

        // Earthquake: driven by tectonic stress
        for (TectonicPlate plate : planet.plates()) {
            if (plate.stress() > 0.8f && rand.nextFloat() < freq * 0.05f) {
                BlockPos epicenter = choosePlateBoundaryEpicenter(plate, rand);
                DisasterInstance quake = new DisasterInstance(
                    DisasterType.EARTHQUAKE, epicenter, plate.stress());
                boolean cancelled = NeoForge.EVENT_BUS.post(new DisasterStartEvent(level, quake)).isCanceled();
                if (!cancelled) {
                    planet.addDisaster(quake);
                    plate.releaseStress(0.6f);
                }
            }
        }

        // Wildfire: driven by drought + temperature
        if (roll < freq * 0.02f) {
            ClimateSnapshot snap = ClimateEngine.getSnapshot(level, BlockPos.ZERO);
            if (snap.isDry() && snap.isHot()) {
                BlockPos epicenter = new BlockPos(
                    rand.nextInt(10000) - 5000, 64, rand.nextInt(10000) - 5000);
                DisasterInstance fire = new DisasterInstance(
                    DisasterType.WILDFIRE, epicenter, snap.temperature() / 50f);
                if (!NeoForge.EVENT_BUS.post(new DisasterStartEvent(level, fire)).isCanceled()) {
                    planet.addDisaster(fire);
                }
            }
        }
    }

    private static BlockPos choosePlateBoundaryEpicenter(TectonicPlate plate, Random rand) {
        var boundaries = plate.boundaries();
        if (boundaries.isEmpty()) return BlockPos.ZERO;
        var boundary = boundaries.get(rand.nextInt(boundaries.size()));
        int[] xs = boundary.epicenterX();
        int[] zs = boundary.epicenterZ();
        int i = rand.nextInt(xs.length);
        return new BlockPos(xs[i], 64, zs[i]);
    }
}
```

```java
package com.horizonpack.horizonworld.data.ecology;

import com.mojang.serialization.Codec;
import com.mojang.serialization.MapCodec;
import com.mojang.serialization.codecs.RecordCodecBuilder;
import net.minecraft.resources.ResourceLocation;

/**
 * Tracks ecological balance for one biome instance (per chunk cluster).
 * HorizonBiome reads this to drive spawn rates and flora density.
 */
public record EcologyProfile(
        ResourceLocation biomeId,
        float predatorCount,    // normalized 0–1
        float preyCount,        // normalized 0–1
        float plantBiomass,     // normalized 0–1
        float soilHealth,       // 0–1 (drives crop yield multiplier)
        float waterTable,       // 0–1 (0=drought, 1=flooded)
        SoilType soilType
) {
    public static final MapCodec<EcologyProfile> CODEC = RecordCodecBuilder.mapCodec(inst -> inst.group(
            ResourceLocation.CODEC.fieldOf("biome").forGetter(EcologyProfile::biomeId),
            Codec.FLOAT.fieldOf("predator_count").forGetter(EcologyProfile::predatorCount),
            Codec.FLOAT.fieldOf("prey_count").forGetter(EcologyProfile::preyCount),
            Codec.FLOAT.fieldOf("plant_biomass").forGetter(EcologyProfile::plantBiomass),
            Codec.FLOAT.fieldOf("soil_health").forGetter(EcologyProfile::soilHealth),
            Codec.FLOAT.fieldOf("water_table").forGetter(EcologyProfile::waterTable),
            SoilType.CODEC.fieldOf("soil_type").forGetter(EcologyProfile::soilType)
    ).apply(inst, EcologyProfile::new));

    /** Returns imbalance score 0.0 (perfect) to 1.0 (critical). */
    public float imbalanceScore() {
        float predPreyRatio = predatorCount > 0 ? preyCount / predatorCount : 0f;
        float predPreyScore = Math.abs(predPreyRatio - 10f) / 10f; // ideal ratio 1:10
        float biomassScore  = 1f - plantBiomass;
        float soilScore     = 1f - soilHealth;
        float waterScore    = Math.abs(waterTable - 0.5f) * 2f;
        return (predPreyScore + biomassScore + soilScore + waterScore) / 4f;
    }

    public EcologyProfile withPredatorCount(float v)  { return new EcologyProfile(biomeId, v, preyCount, plantBiomass, soilHealth, waterTable, soilType); }
    public EcologyProfile withPreyCount(float v)      { return new EcologyProfile(biomeId, predatorCount, v, plantBiomass, soilHealth, waterTable, soilType); }
    public EcologyProfile withPlantBiomass(float v)   { return new EcologyProfile(biomeId, predatorCount, preyCount, v, soilHealth, waterTable, soilType); }
    public EcologyProfile withSoilHealth(float v)     { return new EcologyProfile(biomeId, predatorCount, preyCount, plantBiomass, v, waterTable, soilType); }
    public EcologyProfile withWaterTable(float v)     { return new EcologyProfile(biomeId, predatorCount, preyCount, plantBiomass, soilHealth, v, soilType); }
```

```java
    private static final float IMBALANCE_THRESHOLD = 0.65f;

    public static EcologyProfile getProfile(int chunkX, int chunkZ) {
        return profiles.get(chunkKey(chunkX, chunkZ));
    }

    public static void updateProfile(int chunkX, int chunkZ, EcologyProfile profile) {
        profiles.put(chunkKey(chunkX, chunkZ), profile);
        if (profile.imbalanceScore() > IMBALANCE_THRESHOLD) {
            // Fired on main thread via server tick, so posting directly is safe
        }
    }

    /** Called on server tick (every 100 ticks) for loaded chunks near players. */
    public static void tickEcology(ServerLevel level) {
        // Iterate through profiles for loaded chunks and apply natural recovery
        profiles.replaceAll((key, profile) -> {
            float newSoil = Math.min(1f, profile.soilHealth() + 0.001f); // slow natural recovery
            float newBiomass = Math.min(1f, profile.plantBiomass() + 0.002f);
            EcologyProfile updated = profile.withSoilHealth(newSoil).withPlantBiomass(newBiomass);

            if (updated.imbalanceScore() > IMBALANCE_THRESHOLD) {
                NeoForge.EVENT_BUS.post(new EcologyImbalanceEvent(level, updated, key));
            }
            return updated;
        });
    }

    private static long chunkKey(int cx, int cz) {
        return (long)cx << 32 | (cz & 0xFFFFFFFFL);
    }
}
```

```java
package com.horizonpack.horizonworld.events;

import com.horizonpack.horizoncore.events.custom.DayAdvanceEvent;
import com.horizonpack.horizonworld.data.climate.SeasonData;
import com.horizonpack.horizonworld.data.planet.PlanetData;
import com.horizonpack.horizonworld.network.WorldPacketHandler;
import com.horizonpack.horizonworld.network.packets.*;

import net.minecraft.server.level.ServerLevel;
import net.minecraft.server.level.ServerPlayer;
import net.neoforged.neoforge.event.entity.player.PlayerEvent;
import net.neoforged.neoforge.event.level.LevelEvent;
import net.neoforged.neoforge.event.tick.LevelTickEvent;
import net.neoforged.bus.api.SubscribeEvent;

public class WorldEventHandlers {
    private static int ecologyAccum = 0;

    @SubscribeEvent
    public static void onServerLevelTick(LevelTickEvent.Post event) {
        if (!(event.getLevel() instanceof ServerLevel level)) return;
        if (level.isClientSide()) return;

        SeasonData seasonData = SeasonData.get(level);
        WeatherEngine.tick(level, seasonData.getWeatherState());
        DisasterEngine.tick(level);
        TectonicEngine.tick(level);

        ecologyAccum++;
        if (ecologyAccum >= 100) {
            ecologyAccum = 0;
            EcologyEngine.tickEcology(level);
        }
    }

    /** Listens to HorizonCore's DayAdvanceEvent to advance seasons. */
    @SubscribeEvent
    public static void onDayAdvance(DayAdvanceEvent event) {
        if (!(event.level() instanceof ServerLevel level)) return;
        SeasonEngine.onDayAdvance(level);
    }

    @SubscribeEvent
    public static void onPlayerLogin(PlayerEvent.PlayerLoggedInEvent event) {
        if (!(event.getEntity() instanceof ServerPlayer player)) return;
        ServerLevel level = (ServerLevel) player.level();
        SeasonData seasonData = SeasonData.get(level);

        // Sync full world state to joining player
        WorldPacketHandler.sendToPlayer(player, new SyncSeasonPacket(
            seasonData.getCurrentSeason(), seasonData.getDayInSeason()));
        WorldPacketHandler.sendToPlayer(player, new SyncWeatherPacket(
            seasonData.getWeatherState()));
        WorldPacketHandler.sendToPlayer(player, new SyncDisasterPacket(
            PlanetData.get(level).activeDisasters()));
    }

    @SubscribeEvent
    public static void onLevelLoad(LevelEvent.Load event) {
        if (!(event.getLevel() instanceof ServerLevel level)) return;
        // Ensure PlanetData and SeasonData are initialized
        PlanetData.get(level);
        SeasonData.get(level);
        ClimateEngine.init();
    }
}
```

## 24. Critical Implementation Notes

### 24.1 Chunk Generator Registration
`HorizonChunkGenerator` must be registered via `DeferredRegister` of `BuiltInRegistries.CHUNK_GENERATOR` **and** its `MapCodec` must be added to the codec registry during `FMLCommonSetupEvent`. Without codec registration, the generator cannot be serialized to `level.dat` and will fail on world load:

```java
// In WorldRegistries.registerChunkGeneratorCodec():
Registry.register(BuiltInRegistries.CHUNK_GENERATOR,
    ResourceLocation.fromNamespaceAndPath(HorizonWorld.MODID, "main"),
    HorizonChunkGenerator.CODEC);
```

The dimension JSON at `data/horizonworld/dimension/overworld.json` must reference this generator type explicitly. Players who install HorizonWorld on an existing world will get hybrid terrain — new chunks use the Horizon generator, old chunks retain vanilla geometry. A startup warning is logged if existing level data does not match.

### 24.2 Planet Seed Independence
`ContinentMap` is generated from `continentSeed` (stored in `WorldGenSettings`), which is **derived from, but not identical to**, the Minecraft world seed. This means the continent layout is stable across sessions. Never re-generate `ContinentMap` after world creation — always load from `PlanetData`. The `PlanetData.get()` call returns the saved map; `ContinentMaskPass` reads from it rather than re-running the Voronoi algorithm.

### 24.3 Thread Safety in ChunkGenerator
`fillFromNoise` is called on worker threads from `Util.backgroundExecutor()`. All objects it touches must be either:
- **Immutable** (`WorldGenSettings`, `GeologyProfile`, `ContinentMap`)
- **Thread-local** (per-call `RandomSource` instances)
- **Read-only after init** (`ContinentMap` cells are written only during world creation, then frozen)

Never call `PlanetData.get(level)` from within `fillFromNoise` — `SavedData` access requires the server thread. Instead, pass the `ContinentMap` reference into the pass constructors before the worker threads start.

### 24.4 Weather Suppression of Vanilla Rain
Vanilla Minecraft's rain system (`ServerLevel.setWeatherParameters`) will conflict with the Weather Engine. Use a Mixin on `ServerLevel.tickChunk` and `ClientLevel.getRainLevel` to read from `WeatherClientState` instead of vanilla weather flags. The Mixin targets are:
- `MixinClientLevel.getRainLevel()` — returns intensity from `WeatherClientState`
- `MixinLevelRenderer.renderRain()` — replaced with `WeatherRenderer.render()` which dispatches on `WeatherType`
- `MixinWeatherEffects` — suppresses the vanilla rain/thunder sound loop so `WeatherSoundManager` can manage ambient sounds

Set `level.setWeatherParameters(0, 0, false, false)` once on world load to ensure vanilla rain is permanently suppressed server-side.

### 24.5 Disaster Performance Budget
Each `DisasterInstance` at `PEAK` phase runs `applyPeakEffects()` every tick. With up to 3 simultaneous disasters (configurable), this is at most 3 × 5–10 block-set operations per tick — acceptable. However, `DisasterEngine.maybeSpawnDisaster()` runs only every `CHECK_INTERVAL` (200) ticks and iterates all tectonic plates. Keep `plate.boundaries()` lists short (≤ 10 per plate) or cache the boundary epicenter list.

The aftermath phase (`applyAftermath()`) must complete in a single tick. Pre-compute the list of positions to modify during `WANING` phase and store it in `DisasterInstance`. The aftermath method then drains the precomputed list in batches of 50 blocks per tick until empty, then transitions to `FINISHED`.

### 24.6 ClimateCache Invalidation Strategy
`ClimateEngine` uses a chunk-keyed LRU cache. Invalidation triggers:
- **Season change** (`SeasonChangeEvent`) — full cache wipe via `invalidateAll()`
- **Disaster aftermath** (`DisasterEndEvent`) — invalidate all chunks within `effectRadius / 16` of epicenter
- **Player-placed/broken blocks above Y=100** — do not invalidate (elevation approximation tolerance is ±4 blocks)
- **Permafrost thaw** — invalidate the specific chunk

The cache is `ConcurrentHashMap`-backed with an explicit capacity limit of 4096 entries. `ClimateCache.get()` and `put()` are synchronized at the bucket level. Do not call `ClimateEngine.getSnapshot()` during `fillFromNoise()` (worker thread) — use the direct latitude computation in `ElevationPass` for climate hints during generation.

### 24.7 Biome Source and Climate Mapping
`HorizonBiomeSource` overrides `getNoiseBiome(x, y, z, sampler)`. It reads the `ContinentMap` for the (x, z) position, determines the `ClimateZone` via `LatLonUtil.toLatLon()`, then looks up the biome using a `ClimateZone → List<Biome>` map weighted by elevation and humidity noise. The biome list is built during `FMLCommonSetupEvent` from all registered biomes that carry a `climate_zone` JSON field (see §17.3).

HorizonBiome registers its flora/fauna biomes using this same system — it adds entries to the `ClimateZone → biome` weighting table via HorizonWorld's `BiomeZoneRegistry.registerBiome(climateZone, biomeHolder, weight)` API.

### 24.8 Ecology Profile Persistence
`EcologyEngine.profiles` is an in-memory `ConcurrentHashMap` and is **not persisted** between sessions. On world load, profiles are recomputed lazily on first chunk access. Initial values are computed from:
- Biome's base ecology template (defined in the biome JSON as `"ecology_base"` extension fields)
- Entity count scan on chunk load (`ChunkEvent.Load`) — counts living entities in the chunk and normalizes to predator/prey ratios

This is intentional: ecology state that was "good" when the player left will reset to the biome default on reload. If you need persistence, add `EcologyProfile` serialization to `PlanetData` using a chunk-key → NBT map.

### 24.9 Adding New Disaster Types from Sub-Mods
Sub-mods can add disaster types by:
1. Extending `DisasterType` — not possible (it's an enum). Instead, sub-mods fire `DisasterStartEvent` themselves with a `DisasterInstance` whose type is one of the 9 core types, customized via severity and epicenter.
2. For truly new visual/mechanical disasters, listen to `DisasterPhaseEvent` for `ACTIVE` and `PEAK` phases and add custom block-set logic. `DisasterEngine` will handle phase progression automatically.
3. Register a precondition checker via `DisasterEngine.registerPreconditionChecker(DisasterType, BiPredicate<ServerLevel, PlanetData>)` if the new disaster needs custom geographic conditions.

### 24.10 Data Generation
`DataGenerator` classes for HorizonWorld:
- `ContinentDefinitionProvider` — generates JSON for all 7 default continents
- `GeologyProfileProvider` — generates JSON for all 7 geology profiles
- `WorldBlockStateProvider` — generates blockstate + model JSONs for all 11 world-specific blocks
- `WorldLootTableProvider` — generates loot tables for all 11 blocks
- `WorldTagProvider` — generates `geological_surface`, `disaster_blocks`, `flammable_terrain`, and `c:` namespace tags
- `WorldAdvancementProvider` — generates all 10 advancements

Run `./gradlew runData` before building the jar. Generated resources land in `src/generated/resources/` which is included in the `sourceSets.main.resources` source set.
